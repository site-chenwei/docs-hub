# 仓储管理

更新时间：2021-06-10 00:02:20

**一、 简介**

仓储管理是商家进行商品管理的重要能力，通过仓储API商家可以进行仓库管理和货品管理。

**二、 仓库管理**

![](https://t16img.yangkeduo.com/perse-open-api/2020-06-28/3df9277a-aeef-4c96-ae6c-916cb29a2cc1.png)

注：增加仓库和修改仓库都需要获取地区/国家地址库列表，需通过物流API pdd.logistics.address.get获取。

**三、货品管理：**

![](https://t16img.yangkeduo.com/perse-open-api/2020-06-28/19bbc344-b8ad-4c1a-92b2-f01ea8739b53.png)

货品关联商品sku：

1） 通过pdd.stock.goods.id.to.sku.query查询商品对应sku信息。

2） 通过pdd.stock.ware.sku.update完成货品与商品id、sku的关联。
