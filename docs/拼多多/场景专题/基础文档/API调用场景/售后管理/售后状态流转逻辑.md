# 售后状态流转逻辑

更新时间：2023-10-15 22:39:25

售后类型主要分为4类：退货退款、仅退款、换货、补寄，下文将分别进行售后状态的流转说明：

-   [pdd.refund.agree同意退款](https://open.pinduoduo.com/application/document/api?id=pdd.refund.agree)
    

商家同意退款在以下售后状态可调用同意退款接口：【after\_sales\_status】

【2】买家申请退款，待商家处理

【3】退货退款， 待商家处理

【6】驳回退款，待用户处理

【7】已同意退货退款,待用户发货

【8】平台处理中

【31】已同意拒收退款，待用户拒收

-   [pdd.refund.returngoods.agree商家售后同意退货](https://open.pinduoduo.com/application/document/api?id=pdd.refund.returngoods.agree)
    

商家同意退货在以下售后状态可调用同意退货接口：【after\_sales\_status】

【2】买家申请退款，待商家处理

【6】驳回退款，待用户处理

【8】平台处理中

-   [pdd.refund.exchange.shipping商家换货发货](https://open.pinduoduo.com/application/document/api?id=pdd.refund.exchange.shipping)
    

商家同意换货发货时用此接口回传换货物流单号：【after\_sales\_status】

【3】退货退款，待商家处理

【7】已同意退货退款,待用户发货

【8】平台处理中

### 

1\. 退货退款

在【用户已寄回待商家处理】节点可调用[pdd.nextone.logistics.warehouse.update](https://open.pinduoduo.com/application/document/api?id=pdd.nextone.logistics.warehouse.update)退货入库接口以触发配置的【退货入库自动退】模板，起到一定程度同意退款的作用（非绝对触发）。

![](https://t16img.yangkeduo.com/perse-open-api/2023-10-12/ae2274d4-c81a-451e-9728-6ada0b86d909.png)

### 

2\. 仅退款

在【待商家处理】节点可调用[pdd.rdc.pddgenius.sendgoods.cancel](https://open.pinduoduo.com/application/document/api?id=pdd.rdc.pddgenius.sendgoods.cancel)取消发货接口以触发配置的【取消发货自动退】模板，起到一定程度同意退款的作用（非绝对触发）。
