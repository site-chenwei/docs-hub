# 拼多多云打印编辑平台使用规范

更新时间：2020-06-30 15:12:01

拼多多云打印编辑平台是由拼多多官方推出的，能让使用拼多多电子面单的商家与ISV更快捷地进行电子面单模板编辑的效率工具。本平台提供两种操作人角色：商家和ISV。为了让ISV尽快接入本平台，并对其用户开放本平台功能，请认真阅读以下操作指导！

**功能介绍**

服务商使用拼多多开放平台账号登录平台后，需要先选择该账号下的一个应用（可用）进行操作

ISV预设模板（自定义区）

ISV可以预先为其用户编辑好一个自定义区模板，可以自由添加文本、图形、表格、图片、码式以及ISV预设的打印项，编辑完成并发布后，其用户在商家侧编辑平台的ISV预设模板（自定义区）菜单里即可使用该模板进行编辑/发布。

#### 

**打印项**

ISV可以自由配置电子面单模板打印项，可以自由添加文本、图形、表格、码式。其中，文本和码式的value在实际打单作业中是需要动态传值的（打印报文的自定义区data），所以文本与码式的文本内容请按照以下格式编写：

例：打印报文中自定义区data格式

Plain Text收起

{

 "data": {

 "key1": "value1",

 "key2": "value2",

 "key3": "value3"

 },

 "templateURL": "http://pos-file.yangkeduo.com/express-common-no-cache/common/xmltemplate/printtemplate20190504220932\_648.xml"

}

则对应文本（码式）若需要打印出对应的value1、value2、value3，则文本内容需要填写

<%=data.key1%>、<%=data.key2%>、<%=data.key3%>

注：“key1”、“key2”、“key3”可以是中文，例如：“商品编码”、“商品规格”、“商品数量”

**功能介绍**

商家使用拼多多店铺主账号登录平台后，需要先选择已授权的一个ISV进行操作

1、商家自定义区

商家可以自由创建指定快递公司的自定义区模板：可以自由添加文本、图形、表格、图片、码式以及ISV预设的打印项，编辑完成并发布后会生成自定义区模板链接（也即ISV传给拼多多打印组件的一份打印报文中的第二个templateUrl）。商家在指定的ISV操作打单时，ISV需要通过拼多多开放平台接口获取商家发布的模板链接，并按照打印报文标准格式传入拼多多打印组件。

2、ISV预设模板（自定义区）

商家可以使用ISV预先配置好的自定义区模板进行编辑，可以自由添加文本、图形、表格、图片、码式以及ISV预设的打印项，编辑完成并发布后会生成自定义区模板链接（也即ISV传给拼多多打印组件的一份打印报文中的第二个模板链接）。商家在指定的ISV操作打单时，ISV需要通过拼多多开放平台接口获取商家发布的模板链接，并按照打印报文标准格式传入拼多多打印组件。
