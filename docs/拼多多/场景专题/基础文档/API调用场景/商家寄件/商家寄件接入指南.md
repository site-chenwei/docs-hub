# 商家寄件接入指南

更新时间：2025-11-04 23:47:55

# 

一、背景

为降低商家发货成本，提高商家寄件效率，提高商家权益，平台正式推出了【商家寄件】服务。考虑到不少商家在实际运营中使用第三方订单管理或ERP系统，我们同步构建了开放、兼容的生态接入方案，以更好地融入商家现有工作流程。 已支持的商家权益：超低价格（1.88元起）、上门取件、在线结算、全国覆盖、虚假轨迹免赔（特惠版快递）

**官方入口：**

1）商家寄件PC端访问地址：[https://mms.pinduoduo.com/express/expressOnlineNew?from=message\_yy\_mms](https://mms.pinduoduo.com/express/expressOnlineNew?from=message_yy_mms)

三、开发接入

### 

3.1 整体说明

-   整体流程为：商家开通免密支付->查询快递运费->商家寄件->快递公司分配快递员->快递员上门揽收->核实包裹信息和费用->自动从商家免密支付账户中扣除费用
    
-   开通免密支付流程：
    
    -   电脑端：登录拼多多商家后台->发货中心->商家寄件->确认寄件->按提示开通免密支付
        
    -   手机端：登录拼多多商家版APP->搜索“商家寄件”->维护寄件地址信息->选择特惠版物流->确认寄件->按提示开通免密支付
        
-   新增维护寄件地址：
    
    -   电脑端：登录拼多多商家后台->发货中心->商家寄件->新增寄件地址
        
    -   手机端：登录拼多多商家版APP->搜索“商家寄件”->新增维护寄件地址信息
        
-   一个订单只能存在一个未取消寄件单，不支持一单多包，如果多次发起寄件请求会被拦截
    
-   有超过24小时未支付的订单，则不能使用商家寄件，需要商家依据支付失败原因尽快完成支付
    
-   只支持本平台订单发起商家寄件，其他渠道订单不支持发起寄件请求
    
-   快递员取件前是可以取消寄件，取件后是不可以取消寄件
    
-   合单发货场景目前不支持，需要接入方自行适配
    
-   电子面单打印场景，printdata等密文信息会从商家寄件下单接口返回
