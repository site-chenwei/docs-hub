# 商品发布说明

更新时间：2023-06-18 20:20:31

### 

一．获取类目

获取当前店铺可发布商品类目（cat\_id）

涉及接口：

[pdd.goods.authorization.cats 获取当前授权商家可发布的商品类目信息](https://open.pinduoduo.com/application/document/api?id=pdd.goods.authorization.cats)

[pdd.goods.cats.get 商品标准类目接口](https://open.pinduoduo.com/application/document/api?id=pdd.goods.cats.get)（多平台类目映射匹配使用）

#### 

pdd.goods.authorization.cats说明：

1.  入参parent\_cat\_id=0获取所有的可供发布的一级类目
    
2.  叶子类目获取：循环调用此接口，入参parent\_cat\_id= 上一次调用返回结果中选择的类目id，获得下一级类目id，直至获取叶子类目
    

要点：注意使用最子集类目ID才可用于发布商品，需循环调用上述接口直至获取到最子集类目。

### 

二．获取商品基本信息

#### 

1.准备商品标题

a. 中文按照2个字符计算，最多填写30个汉字（60个字符）

b. 商品标题=商品描述+相关属性，需与当前商品的分类、属性一致

c. 错误填写可能会引起流量减少甚至商品下架，请准确填写！

#### 

2.上传商品轮播图/详情图/视频等素材

上传商品发布、销售活动过程中所需的图片、视频等素材。

涉及接口：

a. 图片上传：[pdd.goods.img.upload 商品图片上传接口](https://open.pinduoduo.com/application/document/api?id=pdd.goods.img.upload)

b. 视频上传：[pdd.goods.video.upload 商品视频上传接口](https://open.pinduoduo.com/application/document/api?id=pdd.goods.video.upload)（部分类目可用，需申请）

[pdd.goods.file.info.get 文件空间详情查询](https://open.pinduoduo.com/application/document/api?id=pdd.goods.file.info.get)（查看上传视频的审核状态）

c. 素材上传：（白底图、长图等，部分类目可用）

-   通过[pdd.goods.filespace.image.upload](https://open.pinduoduo.com/application/document/api?id=pdd.goods.filespace.image.upload)上传图片到图片空间。图片空间限制5G，上传后在商家后台图片空间可见
    
-   通过[pdd.goods.material.create](https://open.pinduoduo.com/application/document/api?id=pdd.goods.material.create)接口上传白底图长图等素材到商品
    
-   通过[pdd.goods.material.delete](https://open.pinduoduo.com/application/document/api?id=pdd.goods.material.delete)接口删除商品中的素材
    
-   通过[pdd.goods.material.query](https://open.pinduoduo.com/application/document/api?id=pdd.goods.material.query)接口查询商品素材列表

3.获取商品属性

准确获取商品必填、可选属性，有利于商品在搜索和推荐中露出。

涉及接口：

[pdd.goods.cat.rule.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.cat.rule.get)类目商品发布规则查询接口【重要！商品发布各项规则均从此接口获取】

[pdd.goods.template.property.value.search](https://open.pinduoduo.com/application/document/api?id=pdd.goods.template.property.value.search) 模板属性值搜索接口

[pdd.goods.out.property.mapping.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.out.property.mapping.get) 站内外属性映射接口

涉及标品：

1.根据类目id查该类目下的标品规则。[pdd.goods.cat.rule.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.cat.rule.get)

2.根据标品搜索接口模糊搜索标品。[pdd.goods.spu.search](https://open.pinduoduo.com/application/document/api?id=pdd.goods.spu.search)

3.根据标品查询接口精确获取标品信息。[pdd.goods.spu.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.spu.get)

4.商品发布时将获取的标品信息传到商品属性信息上。

具体操作：

通过[pdd.goods.cat.rule.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.cat.rule.get)查询该类目的商品发布需要的属性及其他发布规则

1.  通过叶子类目id获取该类目的发布规则，目前返回标品、商品服务、属性等规则。
    
2.  从返回参数中获取ref\_pid、vid（输入型属性值无此参数）、value入参到商品上传或编辑接口的goods\_properties字段。
    
3.  注：接口返回的属性中is\_sale=true的为销售属性（标准属性），入参此类属性需与规格结合，直接从该接口获取spec\_id（规格id）并同时入参到属性字段goods\_properties与规格字段sku\_list中的spec\_id\_list；

#### 

4.获取商品规格

设置商品销售的规格，一般规格至多2种（销售属性规格可增至4种）

涉及接口：

[pdd.goods.spec.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.spec.get) 商品属性类目接口

[pdd.goods.spec.id.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.spec.id.get) 生成商家自定义的规格

注意：

a. 通过[pdd.goods.spec.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.spec.get)接口获取对应要发布的商品所属叶子类目所需的规格id

b. 根据获取到的生成规格属性id请使用[pdd.goods.spec.id.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.spec.id.get)接口生成商家自定义规格

c. 部分类目已切至新版属性发布，请使用[pdd.goods.cat.rule.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.cat.rule.get)获取到的规格值（标准规格is\_sale=true）发布。

一般规格类型不超过2（销售属性规格至多为4种），spec\_id\_list中入参不超过2个spec\_id；

“一般类目规格和不超过44”指所有规格值不超过44，如颜色和尺寸：“黑、白....”加上“X、XL....”总数不超过44。

所有规格在spec\_id\_list中需按笛卡尔积入参，即是需要将两种规格两两组合，举例：若有颜色A、B和尺寸a、b则有\[A，a\]\[A，b\]\[B，a\]\[B，b\]四条sku。

#### 

5.设置商品价格及库存

商品上传时需预设商品参考价格及各规格单买价、拼单价及库存等信息。

**注意要点：**

价格：

-   参考价格应高于所有SKU中的最大单买价格。
    
-   SKU单买价格需要比拼单价格高至少一元。
    

库存：

-   预估当前库存加上库存增减值应该大于或等于0
    
-   下架的sku不能修改库存
    
-   商品在资源位不能减少库存
    

#### 

6.服务与承诺设置

设置商品类型、是否二手、预售信息、承诺发货时间、运费模板、拼单要求等。

预售信息：预售时间不能小于当前时间+承诺发货时间+1天之和

承诺发货时间：

-   普通商品：当日发货、24小时、48小时
    
-   进口商品：48小时
