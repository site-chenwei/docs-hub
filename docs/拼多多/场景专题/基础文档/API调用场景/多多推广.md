# 多多推广

更新时间：2020-06-30 15:12:01

### 

**一、 简介**

多多推广api是专门服务于商家用于商品推广的接口，目前支持【搜索推广】与【场景推广】，通过这些api可以进行推广计划、推广单元、推广定向、关键词、创意的查询与管理，还可以查询相应的推广数据报表。

目前多多推广API对接口调用有限定配额，如遇问题，请联系推广服务商邮件组：ad-isv@pinduoduo.com

### 

**三、 操作流程**

#### 

**查询**

1、 查询开户与详情信息。可调用pdd.ad.api.advertiser.query.account.info和pdd.ad.api.advertiser.query.detail接口查询商家推广开户信息和广告主详情信息。

2、 如未开户，操作开户。可调用pdd.ad.api.advertiser.open.account接口协助商家进行广告主开户。

3、 如已开户，查询推广账户余额。保证商家的拼多多推广账户中有足够的可用金额，可调用pdd.ad.api.advertiser.query.account.balance 接口查询商家推广账户余额，如果推广账户没有余额则需要引导商家到商家后台进行充值。

4、 查询推广计划列表。可调用pdd.ad.api.plan.query.list接口查询推广计划列表。

5、 查询推广单元列表。可调用pdd.ad.api.unit.query.list接口查询推广单元列表。

6、 查询可用商品列表。可调用pdd.ad.api.goods.query.page接口查询可用商品列表。

7、查询定向/资源位列表。可调用pdd.ad.api.unit.bid.query.list接口查询定向/资源位列表。可通过pdd.ad.api.unit.bid.query.targeting.tag.list接口获取获取定向标签数据。可通过pdd.ad.api.unit.bid.query.base.target.profile接口获取可用基础定向，pdd.ad.api.unit.bid.query.base.location.profile（仅场景）接口获取可用资源位， pdd.ad.api.unit.bid.query.interest.profile（仅场景）接口获取可用兴趣点定向。也可通过pdd.ad.api.unit.bid.query.audience.profile接口获取DMP人群包。

8、查询关键词列表。可调用pdd.ad.api.keyword.query.list接口查询关键词列表。

9、查询创意列表。可调用pdd.ad.api.unit.creative.query.list接口查询创意列表。

10、查询智能创意流量分配比例。可调用pdd.ad.api.unit.creative.query.flow.rate接口查询查询智能创意流量分配比例（万分比）。

11、查询商品长图。可调用pdd.ad.api.goods.query.long.images接口查询商品长图。

12、查询商品轮播图。可调用pdd.ad.api.goods.query.gallery.images接口查询商品轮播图。

#### 

**创建**

1、 创建推广计划条件验证。首先使用pdd.ad.api.plan.query.can.create.ad.plan接口进行创建推广计划的条件验证，创建推广计划需商家同意推广软件服务协议且计划数在50以下。

2、 创建推广计划。使用pdd.ad.api.plan.create 接口创建推广计划。

3、 创建推广单元。使用pdd.ad.api.unit.create接口创建推广单元，同一个推广计划下最多可以创建20个推广单元，包含设定出价、定向（见人群/资源位定向查询）、智能创意。在创建推广单元时，可调用pdd.ad.api.goods.query.page接口查询可用商品列表页选定目标推广商品。

4、 （仅搜索）推广单元添加关键词。需要通过pdd.ad.api.keyword.create接口创建关键词对推广单元进行关联。也可通过pdd.ad.api.keyword.recommend.get接口获取系统推荐的关键词进行添加。

5、 推广单元增加创意。通过pdd.ad.api.unit.creative.create接口给推广单元增加创意，并通过pdd.ad.api.unit.creative.check.title接口检查创意标题是否合法。在创建创意时，可调用pdd.ad.api.goods.query.long.images接口查询商品长图，pdd.ad.api.goods.query.gallery.images接口查询商品轮播图。

6、 此时推广计划已经创建完毕，调用pdd.ad.api.plan.update.data.operate.status、pdd.ad.api.unit.update.data.operate.status和pdd.ad.api.unit.creative.update.data.operate.status接口可完成推广计划、推广单元、创意的开启或暂停。

#### 

**删除**

1、 通过pdd.ad.api.plan.delete接口删除推广计划。删除推广计划前需删除这个计划下的全部推广单元。

2、 通过pdd.ad.api.unit.delete接口删除推广单元。

3、 通过pdd.ad.api.keyword.delete接口删除关键词。

4、 通过pdd.ad.api.unit.bid.delete接口删除定向/资源位。

5、 通过pdd.ad.api.unit.creative.delete接口删除创意。注意，每个推广单元至少要保留一个创意。

#### 

**修改**

1、 通过pdd.ad.api.plan.update.plan.name接口修改推广计划名称。

2、 通过pdd.ad.api.plan.update.max.cost接口修改推广计划日限额。修改推广计划日限额前，需通过pdd.ad.api.plan.query.can.update.max.cost接口确认当天是否允许更新计划的推广预算。

3、 通过pdd.ad.api.plan.update.discount 接口修改推广计划分时折扣。

4、 通过pdd.ad.api.plan.update.data.operate.status接口修改推广计划状态（暂停、开启）。

5、 通过pdd.ad.api.unit.update.unit.name接口修改推广单元名称.

6、 通过pdd.ad.api.unit.update.unit.bid接口修改推广单元出价。

7、 通过pdd.ad.api.unit.update.optimization.message接口修改优化工具模块的参数（如OCPX的出价）。

8、 通过pdd.ad.api.unit.update.data.operate.status接口修改推广单元状态（暂停、开启）。

9、 通过pdd.ad.api.unit.bid.sync 接口同步更新定向/资源位。

10、 通过pdd.ad.api.unit.bid.update接口修改更新单个定向/资源位设定。

11、 通过pdd.ad.api.keyword.update接口修改搜索推广的关键词。

12、 通过pdd.ad.api.unit.creative.update.content接口修改创意内容。

13、 通过pdd.ad.api.unit.creative.update.smart.creative接口修改智能创意。

14、 通过pdd.ad.api.unit.creative.distribute.flow.rate接口修改智能创意流量比例分配。

15、 通过pdd.ad.api.unit.creative.update.data.operate.status接口修改创意状态（暂停、开启）。

#### 

**查看推广数据报表**

1、查询分天报表数据

通过pdd.ad.api.report.daily.report.query 接口可按天查询投放数据，查询主体包含广告主、计划、单元、定向、创意、资源位、关键词，查询场景包含搜索、场景展示。返回数据按天分组，即一天一条数据。

2、查询小时报表数据

通过pdd.ad.api.report.hourly.report.query 接口可按小时查询投放数据，查询主体包含广告主、计划、单元、定向、创意、资源位、关键词，查询场景包含搜索、场景展示。返回数据按小时分组，即1小时一条数据。

3、查询分级报表数据

通过pdd.ad.api.report.entity.report.query 接口可查询分级数据，查询主体包含广告主、计划、单元、定向、创意、资源位、关键词，查询场景包含搜索、场景展示。Eg：入参主体维度entityId 为计划1，查询维度entityDimensionType为单元，时间为1-10号，则返回计划1下所有单元1-10号的汇总数据，返回数据按单元分组，即每个单元一条数据。
