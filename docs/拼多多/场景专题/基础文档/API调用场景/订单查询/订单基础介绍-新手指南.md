### 

应用创建

认证角色完成之后，进入 应用列表，点击 创建应用，选择需创建的应用类型（不同角色申请应用类型种类不同），目前不同应用类型创建审核需要提交的材料不同，请以实际申请的页面为准

创建审核通过后应用处于测试中的状态，此时开发者已经可以对软件进行开发测试。但该阶段授权商家数和接口调用数量都会受到限制，且受应用时限管控，请于功能开发完成后尽快将应用提交上线审核，以免影响后续入驻流程

按您产品功能需求，合理选择当前开放可申请的应用类型

### 

应用上线

当软件功能测试无误后，请立即发布应用上线审核，按现行规则，对长期未上线应用将被驳回，后续将无法继续调用接口

### 

发布服务

电商软件角色下应用类型需要发布服务市场供商家订购使用，应用上线审核通过后，进入 [服务管理](https://open.pinduoduo.com/service/list) 页面，选择要发布的应用

注：对无需发布服务市场的应用类型请忽略此步骤
