# 发票管理

更新时间：2020-06-30 15:12:01

#### 

**一、简介**

买家申请开具发票后，服务商可以通过接口给商家提供手动或自动开票的功能，服务商通过自己对接的开票平台开完发票后，还需再将相应的票据信息回传给开放平台。

在接入之前，请先确认下，如果是走新的平台自动开票的方式，则不需对接开平、商家直接找招商接入即可。

#### 

**二、名词解释**

申请流水号：当买家申请开票时，系统为该次申请创建的申请编号。

发票流水号：当回传发票后，系统针对该申请流水号创建的开票编号，表示该次申请发票已回传。

#### 

**三、API**

pdd.invoice.application.query（开票申请单查询）

pdd.invoice.detail.upload（开票结果回传）

pdd.invoice.detail.invalid（订单发票冲红）

#### 

**四、操作流程**

1、 通过pdd.invoice.application.query接口查询买家申请的开票的申请单。

2、 通过pdd.invoice.detail.upload接口来对发票进行回传。

3、 通过pdd.invoice.detail.invalid接口可以根据订单号对后台发票进行冲红（作废处理）。
