# 商品发布说明

### 一．获取类目

获取当前店铺可发布商品类目（cat_id）

涉及接口：

[pdd.goods.authorization.cats 获取当前授权商家可发布的商品类目信息](https://open.pinduoduo.com/application/document/api?id=pdd.goods.authorization.cats)

[pdd.goods.cats.get 商品标准类目接口](https://open.pinduoduo.com/application/document/api?id=pdd.goods.cats.get)（多平台类目映射匹配使用）

#### pdd.goods.authorization.cats说明：

- 入参parent_cat_id=0获取所有的可供发布的一级类目

- 叶子类目获取：循环调用此接口，入参parent_cat_id= 上一次调用返回结果中选择的类目id，获得下一级类目id，直至获取叶子类目

要点：注意使用最子集类目ID才可用于发布商品，需循环调用上述接口直至获取到最子集类目。

#### 商品类目校验

| **校验内容 ** | **相应规则报错** |
| --- | --- |
| 校验商品类目是否有叶子类目 | 商品必须有三级或四级分类 |
| 检验商品类目是否属于店铺可发的一级类目 | 提交不成功，当前商品分类不在店铺经营范围内，请重新选择商品分类 |
| 禁止类目切换 | 虚拟商品不允许更换商品分类，若需更换，请重新发布商品 |
| 处方药类目 | 发布该类目商品，需要绑定互联网医院，请您联系对接招商确认 |
|  | 发布该类目商品，需要同意《处方药售卖补充协议》，请前往mms.pinduoduo.com编辑商品，阅读并同意协议 |

### 二．获取商品基本信息

#### 1.准备商品标题

a. 中文按照2个字符计算，最多填写30个汉字（60个字符）

b. 商品标题=商品描述+相关属性，需与当前商品的分类、属性一致

c. 错误填写可能会引起流量减少甚至商品下架，请准确填写！

#### 2.上传商品轮播图/详情图/视频等素材

上传商品发布、销售活动过程中所需的图片、视频等素材。

涉及接口：

a. 图片上传：[pdd.goods.img.upload 商品图片上传接口](https://open.pinduoduo.com/application/document/api?id=pdd.goods.img.upload)

b. 视频上传：[pdd.goods.video.upload 商品视频上传接口](https://open.pinduoduo.com/application/document/api?id=pdd.goods.video.upload)（部分类目可用，需申请）

[pdd.goods.file.info.get 文件空间详情查询](https://open.pinduoduo.com/application/document/api?id=pdd.goods.file.info.get)（查看上传视频的审核状态）

c. 素材上传：（白底图、长图等，部分类目可用）

- 通过[pdd.goods.filespace.image.upload](https://open.pinduoduo.com/application/document/api?id=pdd.goods.filespace.image.upload)上传图片到图片空间。图片空间限制5G，上传后在商家后台图片空间可见
- 通过[pdd.goods.material.create](https://open.pinduoduo.com/application/document/api?id=pdd.goods.material.create)接口上传白底图长图等素材到商品
- 通过[pdd.goods.material.delete](https://open.pinduoduo.com/application/document/api?id=pdd.goods.material.delete)接口删除商品中的素材
- 通过[pdd.goods.material.query](https://open.pinduoduo.com/application/document/api?id=pdd.goods.material.query)接口查询商品素材列表

各业务素材要求说明如下：

| **业务线** | **素材** | **尺寸** | **文件大小** |
| --- | --- | --- | --- |
| 商品编辑 | 轮播图 | 等宽高 480px以上 | 1M |
|  | 轮播视频 | 时长60s以内 尺寸比例16：9、1：1或3：4 | 300M以内 |
|  | 商详视频 | 时长180s以内，尺寸比例16：9 | 300M以内 |
|  | 商详图 | 宽高比>1：3，宽480px以上 | 1M |
|  | sku图 | 宽高比为1:1，宽高大于480px | 1M |
| 商品素材 | 白底图、场景图、模特白底图、透明图 | 宽高比为1:1，宽高大于480px | 3M |
|  | 长图 | 宽高比为2:3，宽大于400px，高大于600px | 3M |
|  | 带货视频 | 时长60s以内，宽高比为1:2到2:1之间 | 300M |
|  | 短标题 | 6-20个字符 |  |

#### 3.获取商品属性

准确获取商品必填、可选属性，有利于商品在搜索和推荐中露出。

涉及接口：

[pdd.goods.cat.rule.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.cat.rule.get)类目商品发布规则查询接口【重要！商品发布各项规则均从此接口获取】

[pdd.goods.template.property.value.search](https://open.pinduoduo.com/application/document/api?id=pdd.goods.template.property.value.search) 模板属性值搜索接口

[pdd.goods.out.property.mapping.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.out.property.mapping.get) 站内外属性映射接口

涉及标品：

1.根据类目id查该类目下的标品规则。[pdd.goods.cat.rule.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.cat.rule.get)

2.根据标品搜索接口模糊搜索标品。[pdd.goods.spu.search](https://open.pinduoduo.com/application/document/api?id=pdd.goods.spu.search)

3.根据标品查询接口精确获取标品信息。[pdd.goods.spu.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.spu.get)

4.商品发布时将获取的标品信息传到商品属性信息上。

具体操作：

通过[pdd.goods.cat.rule.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.cat.rule.get)查询该类目的商品发布需要的属性及其他发布规则

- 通过叶子类目id获取该类目的发布规则，目前返回标品、商品服务、属性等规则。

- 从返回参数中获取ref_pid、vid（输入型属性值无此参数）、value入参到商品上传或编辑接口的goods_properties字段。

- 注：接口返回的属性中is_sale=true的为销售属性（标准属性），入参此类属性需与规格结合，直接从该接口获取spec_id（规格id）并同时入参到属性字段goods_properties与规格字段sku_list中的spec_id_list；

标品相关校验（GoodsSpuValidate）

| 内容 |  | 判断条件 | 报错信息 |
| --- | --- | --- | --- |
| spu逻辑相关校验 |  | 校验上传的商品信息是否符合平台的标品信息 | SPU检查未通过，请在MMS更改后重试，原因：XXX |
| 标品类目相关校验 |  | 检查是否需要校验 1.当前编辑的草稿类目是否命中了标品管控/不可换品类目 2.线上商品类目是否命中标品管控/不可换品类目 |  |
|  | 管控类目标品校验 | 编辑商品时不可切换类目 | 此类商品发布成功后不可修改类目和关键属性 |
|  | 不可换品类目标品校验 | 1．编辑线上商品时不可更改类目 2．编辑商品时不可修改关键属性除非关键属性为空且未命中标品 | 此类商品发布成功后不可修改类目和关键属性 |

#### 4.获取商品规格

设置商品销售的规格，一般规格至多2种（销售属性规格可增至4种）

涉及接口：

[pdd.goods.spec.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.spec.get) 商品属性类目接口

[pdd.goods.spec.id.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.spec.id.get)生成商家自定义的规格

注意：

a. 通过[pdd.goods.spec.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.spec.get)接口获取对应要发布的商品所属叶子类目所需的规格id

b. 根据获取到的生成规格属性id请使用[pdd.goods.spec.id.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.spec.id.get)接口生成商家自定义规格

c. 部分类目已切至新版属性发布，请使用[pdd.goods.cat.rule.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.cat.rule.get)获取到的规格值（标准规格is_sale=true）发布。

一般规格类型不超过2（销售属性规格至多为4种），spec_id_list中入参不超过2个spec_id；

“一般类目规格和不超过44”指所有规格值不超过44，如颜色和尺寸：“黑、白....”加上“X、XL....”总数不超过44。

所有规格在spec_id_list中需按笛卡尔积入参，即是需要将两种规格两两组合，举例：若有颜色A、B和尺寸a、b则有[A，a][A，b][B，a][B，b]四条sku。

#### 5.设置商品价格及库存

商品上传时需预设商品参考价格及各规格单买价、拼单价及库存等信息。

**注意要点：**

价格：

- 参考价格应高于所有SKU中的最大单买价格。
- SKU单买价格需要比拼单价格高至少一元。

库存：

- 预估当前库存加上库存增减值应该大于或等于0
- 下架的sku不能修改库存
- 商品在资源位不能减少库存

#### 6.服务与承诺设置

设置商品类型、是否二手、预售信息、承诺发货时间、运费模板、拼单要求等。

预售信息：预售时间不能小于当前时间+承诺发货时间+1天之和

承诺发货时间：

- 普通商品：当日发货、24小时、48小时
- 进口商品：48小时
- 直邮直供：5天
- cc行邮：5天

运费模板：[pdd.goods.logistics.template.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.logistics.template.get)商品运费模版接口

商品服务承诺校验

| **内容** | **判断条件** | **报错信息** |
| --- | --- | --- |
| 七天无理由退换货（isRefundable） | 只能选7天无理由退货规则 | 该分类下只能选择7天无理由退货 |
|  | 必不能选7天无理由退货规则 | 该分类下不能选择7天无理由退货 |
| 假一罚十（isFolt） | 必选假一罚十规则 | 该类目下假一罚十必选 |
|  | 不显示假一赔十规则 | 该类目下假一罚十必须选否 |
| 二手商品（secondHand） | 是否二手商品必须选择否 | 是否二手商品必须选择否 |
|  | 是否二手商品必须选择是 | 是否二手商品必须选择是 |
|  | 二手商品不允许选择海外进口、海外BC直邮、保税仓BC直供 | 二手商品不允许选择海外进口、海外BC直邮、保税仓BC直供 |

|  | 必选缺重包退规则 | 必选缺重包退 |

### 三．发布商品

#### 1.直接发布上架

将准备好的商品参数信息入参到[pdd.goods.add](https://open.pinduoduo.com/application/document/api?id=pdd.goods.add)商品新增接口提交发布。

#### 2.创建草稿并从草稿发布上架

[pdd.goods.edit.goods.commit](https://open.pinduoduo.com/application/document/api?id=pdd.goods.edit.goods.commit) 新增或编辑草稿接口

[pdd.goods.submit.goods.commit](https://open.pinduoduo.com/application/document/api?id=pdd.goods.submit.goods.commit)编辑并提交草稿接口

| --- | --- | --- |

|  | 是否在限制的数量范围内 | sku数量超过限制 |

### 四．查看发布状态并编辑

使用草稿状态接口查询商品发布状态，根据商品状态信息编辑调整并重新提交。

[pdd.goods.commit.detail.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.commit.detail.get)获取商品详情(单商品单草稿详情查询)

[pdd.goods.commit.list.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.commit.list.get) 草稿列表接口（根据草稿状态批量查询草稿）

[pdd.goods.commit.status.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.commit.status.get)草稿状态查询接口（草稿id批量查询草稿）

[pdd.goods.latest.commit.status.get](https://open.pinduoduo.com/application/document/api?id=pdd.goods.latest.commit.status.get) 批量goodsId查询最新的审核状态

1. 编辑线上商品：

[pdd.goods.information.update](https://open.pinduoduo.com/application/document/api?id=pdd.goods.information.update) 商品编辑

[pdd.goods.quantity.update](https://open.pinduoduo.com/application/document/api?id=pdd.goods.quantity.update)接口修改sku库存

[pdd.goods.sku.price.update](https://open.pinduoduo.com/application/document/api?id=pdd.goods.sku.price.update) 接口更新sku价格

2. 编辑草稿并提交：

[pdd.goods.submit.goods.commit](https://open.pinduoduo.com/application/document/api?id=pdd.goods.submit.goods.commit) 编辑并提交草稿接口
