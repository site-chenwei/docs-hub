# 商品基础介绍

### 商品业务基础说明

- **goods_id 拼多多商品ID**

由纯数字构成，是商品的唯一识别码，可在店铺后台查看。

【商品管理】-【商品列表】-【线上商品】，商品标题上方的数字编号即是商品goods_id。

![图片](https://t16img.yangkeduo.com/perse-open-api/2021-02-07/722a7146-6d1d-48c3-917c-22778b35f4fd.png)

- **SKU （Stock Keeping Unit）商品库存保有单位**

可简单理解为选定的一个可下单规格即是一个SKU。

例如： 下图耳机“黑色-无线”就是一个SKU，图中每一个颜色和型号的组合都是一个SKU。

店铺后台【商品管理】-【商品列表】-找到所需商品，点击修改价格或库存即可看到SKUID。

![图片](https://t16img.yangkeduo.com/perse-open-api/2021-02-07/793cf4e0-aa16-4e0b-be78-0dec482a071a.png)

- **商品价格**

拼多多商品价格分为单买价与拼团价，商品各个SKU均需设置单买价与拼团价。

拼多多采用拼团购买形式，选择拼团价下单，当下单人数到达成团要求时即可以拼团价购买商品，价格较单买价为低。

- **商品库存**

商品库存需以SKU维度进行设置，指该SKU的可销售数量。

平台商品库存是指平台商品SKU级别的库存，相同商品规格不同，如：1件装、10件装等，需分别设置库存，实际进行库存管理时需注意平台库存数与商家实际货品数的关系。

- **商品上下架状态**

商家发布商品可以选择上架或者下架，上架商品买家才可选购；下架的商品会在商品列表-已下架中，买家不可见。

商品各个SKU有单独的上下架状态，商品上架时，各SKU展示情况视各自的上下架状态而定，商品下架时SKU均不展示。
