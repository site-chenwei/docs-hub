# 身份认证

注册成功后，继续选择开发者角色-入驻组织资质：

开放平台根据不同的角色资质类型赋予不同的开放能力。

| 角色类型 | 类型描述 | 必备条件 |
| --- | --- | --- |
| 电商软件服务商 | 为拼多多商家、快团团团长提供打单发货、商品管理、售后管理、仓储系统等单个或多个复合业务场景的电商IT系统支持，典型应用如：ERP系统、WMS系统、打单工具、搬家软件等应用 | 企业 |
| 商家服务商 | 为千万拼多多商家提供客服外包、店铺装修、店铺代运营、内容服务、法财税咨询服务等店铺经营或咨询类服务，典型服务如：客服外包、装修模板、主图设计、法务咨询等纯服务 | 企业 |
| 拼多多商家 | 为有软件开发能力的大型商家提供数据能力支持，当前仅开放给具备招商小二对接的品牌商或大商家 | 拼多多店铺（*有招商小二对接） |

#### **选择入驻组织资质前须知**

| **入驻组织资质** | **资质要求** |
| --- | --- |
| 企业 | 企业要求统一社会信用代码91/93开头、营业执照经营范围包含软件开发（或同等近似类型）。 |
| 商家 | 1.拼多多商家后台已注册店铺且店铺审核通过，并缴纳足额店铺保证金 2.已入驻店铺且有招商小二对接（店铺对接小二请登录商家后台【店铺管理-店铺信息】页面可查看，如此处无小二信息，可尝试联系类目小二/其他招商小二沟通对接意向；） |
| 个人 | 具备大陆居民身份证的合法公民 注意：仅可创建“多多客联盟”应用，用于对接多多进宝业务。请至多多进宝官网了解详情。 |

注意：

一个主体只能入驻一次，资质审核通过后不可注销、删除、变更资质类型。请务必谨慎选择！

填写企业信息/管理员信息，提交后等待T+3个工作日审核即可，审核通过或者驳回会有相应短信通知，可在个人中心-资质信息页面查看审核进度。

后续如需**增加其他角色**可至[个人中心-资质信息](https://open.pinduoduo.com/application/developer)页面，申请新角色，提交后等待T+3个工作日审核即可。
