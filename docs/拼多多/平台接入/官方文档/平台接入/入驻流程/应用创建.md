# 应用创建

资质信息审核通过后，可在【控制台-应用列表】，点击右上角创建应用，选择应用类型

### ** 应用类型选择**

应用是开发者调用平台接口的唯一标识。

针对不同的应用类型平台会赋予不同的开放能力，服务商可以根据不同业务场景来申请不同的应用类型。

**不同应用类型划分请参阅下表：**

| 应用类型 | 应用说明 |
| --- | --- |
| 打单 | 发货场景应用，功能需覆盖订单获取、电子面单的打印、发货管理，用于商家发货效率提升 |
| 进销存 | 综合性电商管理系统，功能需覆盖从采购到入库销售的业务环节，用于中小商家的电商流程管理 |
| 订单处理 | 为商家提供基础订单获取、分析、处理、售后、发货等功能的系统，用于中小商家基础订单处理场景 |
| 企业ERP | 综合性电商管理系统，功能覆盖电商业务中订单、商品、库存、采购、财务等环节，用于中大型商家全流程管理 |
| 跨境企业ERP | 综合跨境电商管理系统，功能覆盖跨境业务中订单、商品、库存、财务等环节，用于跨境商家电商全流程管理 |
| 仓储管理系统 | 仓储配送场景应用，功能覆盖：库存管理、专业的波次策略配置、拣货路径优化，用于大型商家的发货效率提升 |
| 商品优化分析 | 商品管理场景应用，功能需覆盖：商品上下架、商品编辑优化，用于商家的商品侧优化管理 |

### **应用信息审核**

选定应用类型后按页面要求填写应用信息，提交后等待T+2个工作日审核即可，审核通过或者驳回会有相应短信通知，可在控制台-应用列表页面查看审核状态。

注：1.应用名称、图标设置需具有品牌识别性

2.回调地址后续非审核状态可修改，保存后立即生效无需审核，用途详见官网“授权说明”文档

3.prd、mrd文档请按页面提供模板填写，产品demo上传软件界面截图即可

4.请填写有效公司官网

### **应用参数说明**

审核通过后可点开“应用列表”—“查看详情”：

client_id： 平台给该应用颁发的唯一标识。

client_secret：密钥，为接口调用提供安全保障，请勿随意透露给他人。

流量证书：当前状态接口调用频次限制

权限包：该应用开放的业务能力，点击可查看具体接口权限
