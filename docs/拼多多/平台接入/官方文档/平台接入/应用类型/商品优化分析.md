# 商品优化分析

**应用信息审核通过后，可进行测试开发：**

**应用信息审核通过后，需在30天内提交上线，否则应用将被驳回，无法调用接口**

## **类目简述**

搬家上货类型属于商品管理场景应用，用于商家批量管理店铺商品的上下架、图文编辑优化

## **功能要求**

| 功能包括不限于： | 商品编辑（上下架、预售、库存、标题、商品外部编码、运费模板等） |
| --- | --- |
|  | 水印标签模板、活动主题图模板 |
|  | 水印/logo编辑 |
|  | 详情页制作 |
|  | 商品素材编辑 |

## **测试开发**

1. 应用创建成功后可在应用详情——授权管理页面配置pdd开头店铺账号（店铺后台右上角“账号管理”中查看）配置后即可授权该店铺用于测试接口

2. 接口对接流程请参阅开放平台文档“授权说明”和“API调用详解”https://open.pinduoduo.com/application/document/browse?idStr=BD3A776A4D41D5F5

3. API调用场景

建议对接场景：商品查询

### **商品查询**[《商品接入场景》](https://open.pinduoduo.com/application/document/browse?idStr=F5961DC9E3C5F174)

**使用场景**

服务商通过商品明细接口及商品列表接口单个或批量获取店铺内商品信息

**相关API**

pdd.goods.list.get

pdd.goods.detail.get

**操作步骤（以下内容仅供参考）**

1）通过pdd.goods.list.get接口根据相关条件查询当前授权商家符合条件的商品列表

2）使用商品id调用pdd.goods.detail.get获取商品详细信息

### **商品编辑**[《商品接入场景》](https://open.pinduoduo.com/application/document/browse?idStr=F5961DC9E3C5F174)

**使用场景**

服务商通过商品编辑接接口可完整编辑所有商品信息，或使用其他专项接口单独编辑上下架状态、库存、价格等信息

**相关API**

pdd.goods.information.update

pdd.goods.sale.status.set

pdd.delete.goods.commit

pdd.goods.quantity.update

pdd.goods.sku.price.update

**操作步骤（以下内容仅供参考）**

1）调用pdd.goods.information.update接口编辑商品信息

2）调用pdd.goods.sale.status.set接口修改商品上下架状态

3）调用pdd.delete.goods.commit接口删除商品，只能删除下架的商品

4）调用pdd.goods.quantity.update接口修改sku库存，注意，在资源位上的商品不能减少库存

5）调用pdd.goods.sku.price.update接口更新sku价格

### **商品草稿**[《商品接入场景》](https://open.pinduoduo.com/application/document/browse?idStr=F5961DC9E3C5F174)

**使用场景**

服务商通过草稿编辑、上传、删除等接口管理商品草稿，商品草稿可避免直接发布商品，用于批量筹备商品信息，以备需要上货时快速提交

**相关API**

pdd.goods.edit.goods.commit

pdd.goods.submit.goods.commit

pdd.delete.draft.commit

**操作步骤（以下内容仅供参考）**

1）通过pdd.goods.edit.goods.commit接口提交到编辑中的商品状态

2）通过pdd.goods.submit.goods.commit接口从草稿提交到发布

3）通过pdd.delete.draft.commit接口删除草稿

### **素材管理**[《商品接入场景》](https://open.pinduoduo.com/application/document/browse?idStr=F5961DC9E3C5F174)

**使用场景**

用于管理商品素材，通过接口将图片上传至图片空间，再通过素材接口管理商品素材的创建、删除、查询等操作

**相关API**

pdd.goods.filespace.image.upload

pdd.goods.material.create

pdd.goods.material.delete

pdd.goods.material.query

**操作步骤（以下内容仅供参考）**

1）通过pdd.goods.filespace.image.upload上传图片到图片空间

2）通过pdd.goods.material.create接口上传白底图长图等素材到商品

3）通过pdd.goods.material.delete接口删除商品中的素材

4）通过pdd.goods.material.query接口查询商品素材列表
