# 常用工具使用说明

更新时间：2020-10-22 23:17:10

### 

一、测试工具

测试工具是平台提供给服务商的一种无需编码的接口请求方式，通常用于业务验证和接口测试使用。

注意：选择线上环境时，通过此工具调用会直接请求线上数据，作用与通过API调用一样。请谨慎操作。

#### 

1、功能模块介绍：

![](https://testimg.yangkeduo.com/perse-open-api/2020-10-13/2f902661-91fb-40c8-a0d8-5a98bc6026fd.png)

1）选择返回数据格式：服务商可以设置返回的数据格式，目前支持JSON和XML两种。

2）选择要测试的接口：目前不支持测试环境，请统一选择正式环境。

3）选择应用：在这里填写应用的clientid和secret，表示用该应用访问接口。应用信息可到：首页-->控制台-->应用查看详情-->基本信息中进行查找。

4）填写商家授权token：需要授权的接口调用时需填写token信息，token的获取可参考[授权说明](https://open.pinduoduo.com/application/document/browse?idStr=BD3A776A4D41D5F5)。也可通过[授权工具](https://open.pinduoduo.com/application/document/apiTools?utilsType=1)获取。

5）接口字段区：在这里填写的是接口的业务字段，相应字段的解释可到相应的[api文档](https://open.pinduoduo.com/application/document/api)中进行查看

6）API请求参数：点击“提交测试”后，测试工具会把请求的报文拼接成字符串进行访问。是接口请求的原生数据，可供开发时参考。

7）API返回结果：向平台发送请求后，如果接口返回成功，会看到具体的接口返回信息。如果返回失败，可参考右上角的“查看错误信息”查看解决办法。

### 

二、授权工具

使用授权工具可以快速给商家授权获取token以作测试使用，正式使用还请参考[授权说明](https://open.pinduoduo.com/application/document/browse?idStr=BD3A776A4D41D5F5)。

#### 

1、使用流程：

1）填入clientid，点击提交

2）登录，mms店铺，点击授权登录。

3）授权后，会跳转到授权页面，该店铺授权获取的access\_token.

### 

三、错误码查询工具

错误码查询工具主要用于帮助服务商根据通过异常报错快速定位问题。

#### 

1、错误码查询：

只需要在排查工具中输入，API名称、错误码（error\_code）、子错误码（sub\_code），点击查询即可找到对应解决方案。

#### 

2、按完整错误信息查询

直接将报错信息填写到文本区，点击查询即可找到对应解决方案。
