# pdd.ddk.statistics.data.query

## 多多进宝数据统计查询接口

更新时间：2021-12-14 11:06:49

¥免费API

不需用户授权

本接口用于查询特定活动数据，仅限特定渠道可用。

#### 公共请求参数

| 参数名称 | 参数类型 | 是否必填 | 参数描述 |
| --- | --- | --- | --- |
| type | String | 必填 | API接口名称 |
| client\_id | String | 必填 | POP分配给应用的client\_id |
| access\_token | String | 非必填 | 通过code获取的access\_token |
| timestamp | String | 必填 | UNIX时间戳，单位秒，需要与拼多多服务器时间差值在10分钟内 |
| data\_type | String | 非必填 | 响应格式，即返回数据的格式，JSON或者XML（二选一），默认JSON，注意是大写 |

| sign | String | 必填 | API输入参数签名结果，签名算法参考开放平台接入指南第三部分底部。 |

### 请求参数说明

| 参数接口 | 参数类型 | 是否必填 | 说明 |
| --- | --- | --- | --- |
| page | INTEGER | 非必填 | 分页数，默认值: 1 |
| page\_size | INTEGER | 非必填 | 每页结果数，默认值: 20 |
| period\_type | INTEGER | 必填 | 周期类型: 1-每7天，2-自然月 |
| statistics\_type | INTEGER | 必填 | 数据类型: 1-增量补贴数据 |
| time | STRING | 必填 | 查询时间点，格式: "yyyy-MM-dd"。period\_type为1时，查询时间点前7天的数据；period\_type为2时，查询时间点所在自然月的数据。 |

JAVA

CURL

```
package com.pdd.pop.sdk.http.demo;

import com.pdd.pop.sdk.common.util.JsonUtil;
import com.pdd.pop.sdk.http.api.pop.request.PddDdkStatisticsDataQueryRequest;
import com.pdd.pop.sdk.http.api.pop.response.PddDdkStatisticsDataQueryResponse;
import com.pdd.pop.sdk.http.PopClient;
import com.pdd.pop.sdk.http.PopHttpClient;

public class PopClientDemo {

    public static void main(String[] args) throws Exception {

        String clientId = "your clientId";
        String clientSecret = "your clientSecret";
        PopClient client = new PopHttpClient(clientId, clientSecret);

        PddDdkStatisticsDataQueryRequest request = new PddDdkStatisticsDataQueryRequest();
        request.setPage(1);
        request.setPageSize(20);
        request.setPeriodType(1);
        request.setStatisticsType(1);
        request.setTime("2021-12-08");
        PddDdkStatisticsDataQueryResponse response = client.syncInvoke(request);
        System.out.println(JsonUtil.transferToJson(response));
    }
}
```
