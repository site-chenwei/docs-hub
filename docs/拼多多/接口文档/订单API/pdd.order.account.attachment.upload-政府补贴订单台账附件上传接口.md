# pdd.order.account.attachment.upload

## 政府补贴订单台账附件上传接口

更新时间：2026-03-25 11:32:33

¥免费API

必须用户授权

对于享受政府补贴的订单，可通过此接口将订单附件上传到商家后台台账系统
