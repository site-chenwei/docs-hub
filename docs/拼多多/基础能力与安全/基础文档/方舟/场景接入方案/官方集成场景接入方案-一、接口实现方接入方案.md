# 官方集成场景接入方案

更新时间：2025-10-19 23:27:51

## 

一、接口实现方接入方案

### 

接口开发

-   [开放管理 - 官方集成](https://open.pinduoduo.com/application/ark/official) 页面找到本次需对接的场景名称（请联系对接的平台小二添加场景权限，才可展示），点击对应场景的 “接口管理”，选择 “查看需实现 api”；查看接口文档，依据文档 **入参规范** 与 **出参规范** 完成后端接口开发（作为接口实现方，需确保接口功能符合平台规范）。
    

![](https://t16img.yangkeduo.com/perse-open-api/2025-10-19/bf3c33b7-24ab-4dcf-8b47-c81aa0da3947.png)

![](https://t16img.yangkeduo.com/perse-open-api/2025-10-19/bad44b07-c553-49d4-b7c9-19019660cbc2.png)

**注意事项：**后端设计需要完全参照 **出参规范** 的参数进行命名。如字段名称对应不上，则不会被方舟网关透传出去。例如**出参规范**中的参数是 is\_success，但是后端返回的结果中的字段为 isSuccess，那么在方舟网关返回的response中将不会有 is\_success 和 isSuccess字段。

### 

接口自测验证

-   接口开发完成后，返回场景接入页面，点击 “接入”进入自测页面；
    

![](https://t16img.yangkeduo.com/perse-open-api/2025-10-19/91927ece-0466-4d15-a136-c231732f4191.png)

-   填写自身后端接口地址，发起自测请求，验证接口是否能正常接收并响应。
    

![](https://t16img.yangkeduo.com/perse-open-api/2025-10-19/8bf7b5c8-256d-43e1-9ad9-eb79e2b65fdd.png)

### 

配置发布

-   自测通过后，点击 “下一步”，填写已通过测试的后端地址，提交 “发布” 申请，发布成功后则全部接入完成。
    

### 

修改已发布的后端地址

[开放管理 - 官方集成](https://open.pinduoduo.com/application/ark/official) 页面点击服务地址管理，可以查看已完成接入的API的后端地址配置，可进行编辑修改

## 

二、接口调用方接入方案

### 

开发与联调

-   [开放管理 - 官方集成](https://open.pinduoduo.com/application/ark/official) 页面找到本次需对接的场景名称（请联系对接的平台小二添加场景权限，才可展示），点击对应场景的 “接口管理”，选择 “查看可调用 api”；查看接口文档，作为调用方，依据文档 **入参规范** 与 **出参规范** 进行开发。
    

-   开发完成后，点击 “联调”，并输入接口入参发起联调；
    

-   点击完成联调。
