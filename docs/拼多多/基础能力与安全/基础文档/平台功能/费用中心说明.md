# 费用中心说明

更新时间：2020-06-30 15:12:01

#### 

一、费用中心入口

入口：[控制台-费用中心](https://open.pinduoduo.com/expense/account)

![](https://t16img.yangkeduo.com/perse-open-api/2020-06-30/953946fd-08cc-4e62-89cc-59c76a803d5f.png)

#### 

二、充值

![](https://t16img.yangkeduo.com/perse-open-api/2020-06-30/8f85d224-6727-4e19-9d3d-f8ead8f9556c.png)

1.  资金账户页，点击充值按钮，输入你需要充值的金额
    
2.  根据提示进行支付，充值成功之后将会在你的可用余额中看到您所充值的金额。
    

#### 

三、扣款

1.  每天凌晨将根据用量生成前一天账单，并从账户余额中自动进行扣款，请保证账户余额充足。
    
2.  若余额不足，将会在下次充值成功之后再次进行扣款。
    

#### 

四、账单数据

入口：[费用中心-账单明细](https://open.pinduoduo.com/expense/account/bill-details)

可以在账单明细页中查看具体的账单明细和调用明细。
