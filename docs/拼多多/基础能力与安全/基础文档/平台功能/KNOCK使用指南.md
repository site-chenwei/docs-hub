# KNOCK使用指南

更新时间：2025-07-29 14:12:49

为了更好的保障商家与服务商的权益，提升开放平台整体工作效率，开放平台工作人员今后将主要通过①拼多多官方的企业办公通讯工具knock以及②工单系统，来提供技术支持与服务，具体使用方式如下：

### 

**1 knock下载方式**

**（****注：下载knock后不要立即注册knock，按照下方的账号使用和注册方法登陆knock即可****）：**

1.1 knock下载地址：[**https://knock.pddim.com/supplier/download**](https://knock.pddim.com/supplier/download)，可根据需要下载PC端、手机客户端；

![](https://testimg.yangkeduo.com/perse-open-api/2020-04-17/35c522d1-52c7-46c9-8b16-5811fe4900ad.png)

1.2 IOS手机用户也可以在应用市场查找“knock企业”进行下载和使用；

![](https://testimg.yangkeduo.com/perse-open-api/2020-04-17/fc030423-084a-436c-b253-961f79160e87.png)

1.3 打开knock之后需要填写手机号与验证码进行登录，登录方式详情见下面的2.knock登录方法；

### 

**2 knock登录方法**

2.1 注册并且通过资质审核的商家或服务商，可以使用注册开放平台的手机号（即主账号）直接登录knock，与拼多多工作人员进行沟通；

2.2 如果公司还有其他的人员有此需要，也可以使用主账号在“开放平台-个人中心-成员管理”模块，创建相应的子账号并为子账号赋予权限（如无权限也可不赋予），保存成功后该子账号绑定的手机号可直接登录knock，与拼多多工作人员进行沟通，具体流程如下图：

-   点击【个人中心-账号管理】进入【成员管理】页面
    

-   点击【添加成员按钮】
    

-   填写成员信息（**注：如果保存时候提示手机号重复，说明系统检测到该手机号之前有注册过开放平台账号，请先用该手机号登陆开放平台然后变更手机号为其他手机号，即释放当前手机号，然后再用该手机号绑定子账号**）
    

### 

**3 knock联系方式**

#### 

3.1 查找小二

登录knock之后，您可以在搜索功能中，搜索对应的拼多多小二花名，与其进行沟通。

注意：多多进宝业务请到https://jinbao.pinduoduo.com/qa-system咨询。

#### 

3.2 接收平台通知

开放平台与服务市场相关的最新消息或紧急通知均会通过“开放平台官方号”及“开放平台通知号”同步至各服务商和商家，因此请保持账号的持续登录状态并关注官方通知

#### 

3.3 企业内部通讯

登录knock之后，您可以在搜索功能中，搜索企业内部其他员工的名称或代号，与其进行日常办公沟通；
