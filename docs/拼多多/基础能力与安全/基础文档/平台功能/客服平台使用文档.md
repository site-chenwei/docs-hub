# 客服平台使用文档

更新时间：2021-04-25 17:56:22

为了提升服务市场商家与服务商的沟通效率，开放平台推出客服平台功能。

商家可在服务市场-服务详情页、订单列表页等入口发起和对应服务商的沟通，服务商在开放平台进行回复。

除客服工具网页版外，当前支持windows客户端版本及手机端版本，请通过[https://mms.pinduoduo.com/other/downloadapp](https://mms.pinduoduo.com/other/downloadapp)下载对应版本，选择服务商账号登录即可。

### 

1.登录入口

使用开放平台账号登录此网页，推荐使用chrome浏览器[https://open.pinduoduo.com/#/index](https://open.pinduoduo.com/#/index)

在【服务管理】-【客服平台】可进入客服工作台。

### 

2.设置接待客服和自动回复

在开放平台右上角-【账号管理】中先创建子账号，再从【客服平台】-【消息分流设置】将子账号设置为接待客服。

接待客服：商家的咨询会随机分配到在线的接待客服账号中；

离线客服：没有客服在线时，分配给离线客服；

语音客服：可从接待客服中，选择一位作为语音客服，接听商家语音通话；

默认主账号为接待客服、离线客服和语音客服（离线客服和语音客服必须从接待客服中选择）；

当公司有多个客服时，可以给客服分配子账号，并在消息分流设置中勾选为接待客服即可。

离线自动回复开启后，无客服在线时，商家会自动收到离线自动回复设置的内容。

### 

3.查看会话列表

客服工作台左侧为会话列表，分为今日会话和全部会话。今日会话展示当天0到24点的商家咨询，每日0点会将前一日的咨询移动到全部会话中。 

### 

4.转移会话

在会话界面，点击右上角的转移会话按钮，即可转移到其他客服。也可以在会话列表，点击批量转移，可以一次性选择10个会话进行转移，被转移的客服可以看到之前会话的历史聊天记录。

### 

5.常见问题

1）怎么在客服平台联系商家？

您可以在【客服平台】中点击“加号”输入商家订单号创建会话与商家聊天。

2）商家在哪里发起咨询的？

服务详情页增加了联系服务商按钮。

3）客服平台可以在多台设备上登录吗？

不可以呢，同一账号可以在不同设备登录开放平台，但是进入客服平台只能为一个账号，当系统识别出该账号在其他设备进入客服平台时，系统会识别“账户在别处登录”，原来客服平台将退出，可以重新登录。

4）如何向商家推荐服务？

您可以在【客服平台】-【服务推荐】中，点击“发送服务”，商家即可收到服务卡片推荐。

5） 客服平台的客服头像在哪里上传？

开发者信息-账号信息中可以上传160\*160的头像

6）为什么有消息发不出去或者被打码？

系统检测出发送不文明用语会进行打码或禁止消息发送，服务商客服不得在沟通过程中使用任何形式的带有人身攻击、侮辱性的不文明用语，不得实施诽谤、谩骂、诋毁商家或使用任何引起商家不满的字句的行为。

7）客服平台网页无法登录，经常掉线，排版异常或卡死，怎么办？ 

账号登录后，登录状态可以保持24小时。使用部分浏览器（如UC浏览器、旧版本的360安全浏览器等）登录客服网页时，如果长时间处于“正在连接服务器”的状态或者其他异常导致长时间无法登录，有以下解决办法：

a）请先检查您的网络是否通畅；

b) Ctrl+F5强制刷新网页；

c) 下载安装最新版本的Chrome浏览器； 

8）如何将客服平台链接加入自己的软件内？

PC端：将Client\_id拼接在链接“https://mms.pinduoduo.com/service-market/chat/isv?client\_id=”中，并将此链接加入软件内，商家可通过此链接在客服平台向服务商发起会话。

移动端：将Client\_id或者Service\_id拼接在链接“https://fuwu.pinduoduo.com/smm-market/chat-isv.html?client\_id=”或“https://fuwu.pinduoduo.com/smm-market/chat-isv.html?service\_id=”中，并将此链接加入软件内，商家可通过此链接在移动端向服务商发起会话。
