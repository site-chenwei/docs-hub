# 解密相关接口

更新时间：2020-12-30 22:35:53

### 

批量加密接口

api: pdd.open.kms.encrypt.batch

https://open.pinduoduo.com/#/apidocument/port?portId=pdd.open.kms.encrypt.batch

说明：可以把本地存量的明文数据替换成加密数据

### 

批量获取搜索文本接口

api: pdd.open.kms.search.batch

https://open.pinduoduo.com/#/apidocument/port?portId=pdd.open.kms.search.batch

说明：可以通过此接口获取搜索值的加密信息用于密文检索

### 

批量解密接口

限制：仅支持云内调用，云外仅支持少量调用供测试使用，云外限流规则1次/10秒

api: pdd.open.decrypt.batch

https://open.pinduoduo.com/application/document/api?id=pdd.open.decrypt.batch

说明：可以通过此接口把获取到的密文信息解密，该接口只支持从多多云内调用，该接口每日有调用次数限制，若非必需明文驱动的业务，请勿全量密文信息都请求该接口解密

### 

批量数据解密脱敏接口

api: pdd.open.decrypt.mask.batch

https://open.pinduoduo.com/application/document/api?id=pdd.open.decrypt.mask.batch

说明：可以通过此接口把密文信息转换成脱敏的内容展示给用户，目前仅支持订单接口的4个字段：receiver\_name、receiver\_phone、receiver\_address 、address；

### 

电子面单云打印接口

api: pdd.waybill.get

https://open.pinduoduo.com/application/document/api?id=pdd.waybill.get

说明：该接口中的消费者敏感信息支持密文入参，即如果无需修改消费者信息字段，取号时可以直接入参获取到的密文字段。

可以直接入参密文的字段：trade\_order\_info\_dtos. recipient. name；trade\_order\_info\_dtos. recipient. phone；trade\_order\_info\_dtos. recipient. address. detail
