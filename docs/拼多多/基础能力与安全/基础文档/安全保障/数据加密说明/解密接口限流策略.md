# 解密接口限流策略

更新时间：2020-09-30 18:14:20

解密限流的开发者应用改造需满足以下：

1.  开发者应用非特殊情况下使用密文获取面单号
    
2.  尽量使用拼多多电子面单，站外面单尽量少使用，减少获取明文的次数
    

### 

解密接口限流策略

1、 假设店铺近一周订单日均发货量是1000单，假设做5%限流，此店铺每天可以解密50个订单

2、 针对一个订单不同字段，如姓名、手机号、地址等将会分别限制解密频率

3、 同一个订单同一天多次获取明文只算一次获取

4、 如果同一个店铺授权多个应用，授权的所有应用共享此店铺的限流次数

5、 如果店铺出现紧急情况 需要扩容流控，请至订单解密管理页面提交提额申请[https://fuwu.pinduoduo.com/service-market/decrypt](https://fuwu.pinduoduo.com/service-market/decrypt)

注：开发者改造初期平台不会默认开启限流，但最终方案会开启上述限流策略
