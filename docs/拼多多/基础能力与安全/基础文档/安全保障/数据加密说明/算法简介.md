# 算法简介

更新时间：2020-12-30 13:47:42

### 

加密算法特点

1.  加密算法使用“AES/CBC/PKCS5Padding”
    
2.  加密用的初始向量会直接编码到加密数据中, 因此相同数据内容多次加密的结果不同
    
3.  不同应用使用不同的秘钥，不同应用间加密数据无法解密
    
4.  支持秘钥轮换, 密文中会包含加密密文用的秘钥版本，更新后支持老版本密文解密
    
5.  密文可以反算出订单号、店铺等信息，如果发现泄漏可追踪

注 3：搜索的输入字符需要至少4个数字或者2个汉字. 如果数据本身长度不够4个数字或2个汉字, 则此时输入全文即可搜索.  如: 原始数据为"你"的, 使用"你"调用平台搜索接口即可获取搜索用文本.

注 4：加密之后的输出均为字符串, 如果原始字段为number类型. 平台会转成string然后加密. 此时api接口会在新字段上添加一个字段, 名为: encrypt\_{原字段名}
