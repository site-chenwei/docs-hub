    if (encryptedData \== null || encryptedData.length() < 4) {

        return null;

    if (encryptedData.charAt(encryptedData.length() \- 2) != sepInData) {

### 

密文格式过渡说明

由于密文中索引串的去除会采用逐步灰度的方案去进行，开发者业务代码需要同时兼容两种密文格式，即部分订单返回的是原始的带索引串的密文，部分订单返回的是不带索引串的密文。

待一段时间灰度后全量切换成新的密文格式（无索引串）。
