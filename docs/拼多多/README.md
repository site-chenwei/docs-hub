# 拼多多文档索引

## 整理说明
- 当前原始文档共 353 篇，其中基础文档 50 篇、官方文档 15 篇、接口文档 288 篇。
- 本轮已新增导航页并同步目录结构，不改各篇文档正文内容。
- 本轮已将 `接口文档` 根目录遗留的 19 篇 API 文档归档到对应业务域目录，并同步补全接口索引。
- 一级分类固定为：平台接入、基础能力与安全、场景专题、接口文档。
- 同主题存在多份文档时，索引只指定一个主入口，其余保留为补充来源。

## 主入口
- [平台接入](./平台接入/README.md)：应用创建、授权、消息推送、SDK、上线审核。
- [基础能力与安全](./基础能力与安全/README.md)：数据安全、平台功能、方舟、服务市场。
- [场景专题](./场景专题/README.md)：商品、订单、售后、寄件、电子面单、虚拟商品等专题说明。
- [接口文档](./接口文档/README.md)：全部 API 域目录与接口清单。

## 建议阅读顺序
1. 先看平台接入，确认应用创建、授权、调用方式和消息推送。
2. 再看基础能力与安全，确认加密、平台功能、方舟与服务市场约束。
3. 按业务场景查场景专题，理解发布、售后、寄件、对账等接入流程。
4. 最后回到接口文档，按具体 API 做参数和出入参核对。
