# 日志规格

#### 应用崩溃

#### \[h2\]CppCrash

可参看CppCrash[日志规格](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cppcrash-guidelines#日志规格)。

#### \[h2\]JSCrash

可参看JS Crash[日志规格](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/jscrash-guidelines#日志规格)。

#### 应用冻屏

可参看应用冻屏（AppFreeze）[日志规格](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/appfreeze-guidelines#日志规格)。

#### 地址越界

可参看地址越界[日志规格](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/address-sanitizer-guidelines#日志规格)。

资源泄漏区分为句柄泄漏、线程泄漏和内存泄漏三类，详见[句柄泄漏日志规格](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/resource-leak-guidelines#section1120134982114)、[线程泄漏日志规格](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/resource-leak-guidelines#section203047817288)、[JS内存泄漏日志规格](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/resource-leak-guidelines#section1827795252818)、[native内存泄漏日志规格](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/resource-leak-guidelines#section8486193719219)和[ashmem/ION/gpu/gpu\_rs内存泄漏日志规格](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/resource-leak-guidelines#section20381028112710)。
