# 创建项目

本章以[Caffe SqueezeNet](https://github.com/forresti/SqueezeNet)模型集成为例，说明App集成操作过程。

1.  创建DevEco Studio项目，选择“Native C++”模板，点击“Next”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c8/v3/L110LOs5SPaHi1hLZfkQtQ/zh-cn_image_0000002538180092.png?HW-CC-KV=V1&HW-CC-Date=20260417T013626Z&HW-CC-Expire=86400&HW-CC-Sign=D4168D11C63F84993307531E238C70AE8459FDAA26BDC59DE98347E185D63EC0)
    
2.  按需填写“Project name”、“Save location”和“Module name”，选择“Compile SDK”为“5.1.0(18)”及以上版本，点击“Finish”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/4b/v3/NqVqPBVdSKCyXNMjSB0AUA/zh-cn_image_0000002569019879.png?HW-CC-KV=V1&HW-CC-Date=20260417T013626Z&HW-CC-Expire=86400&HW-CC-Sign=E9C096D8A9E9798C7B4EE2FBB4F9D9BF46ABD13DD9CCAEB1A4316AE71B6835FE)
