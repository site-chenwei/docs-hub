# 构造函数和析构函数

#### 函数功能

OpRegistrationData构造函数和析构函数。

#### 函数原型

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/27/v3/JeX1nUUITH27qhssyjgpIw/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013635Z&HW-CC-Expire=86400&HW-CC-Sign=77A2DA966FC68194F018F146EDD9CBC4B026E5FB466E3BEF7BD3A0A4AF09A314)

数据类型为string的接口后续版本会废弃，建议使用数据类型为非string的接口。

```cpp
OpRegistrationData(const std::string &om_optype);
OpRegistrationData(const char_t *om_optype);
~OpRegistrationData();
```

OpRegistrationData构造函数返回OpRegistrationData类型的对象。
