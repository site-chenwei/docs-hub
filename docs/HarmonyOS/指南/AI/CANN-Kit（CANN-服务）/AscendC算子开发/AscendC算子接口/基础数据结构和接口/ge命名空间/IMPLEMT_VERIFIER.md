#### 函数原型

```cpp

```

#### 约束说明

无

| 参数名 | 输入/输出 | 描述 |
| :-- | :-- | :-- |
| op\_name | 输入 | 算子类型。 |
| func\_name | 输入 | Verify函数名，开发者自定义。 |

#### 返回值

无
