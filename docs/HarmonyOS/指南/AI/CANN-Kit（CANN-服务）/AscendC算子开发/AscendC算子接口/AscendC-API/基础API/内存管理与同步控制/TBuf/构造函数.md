# 构造函数

#### 函数功能

创建TBuf对象时，初始化数据成员。

#### 函数原型

```cpp
template <TPosition pos> 
__aicore__ inline TBuf();
```

#### 参数说明

| 参数名称 | 含义 |
| :-- | :-- |
| pos | TBuf所在的逻辑位置，取值为VECCALC。关于TPosition的具体介绍请参考[TPosition](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/cannkit-tposition)。 |
