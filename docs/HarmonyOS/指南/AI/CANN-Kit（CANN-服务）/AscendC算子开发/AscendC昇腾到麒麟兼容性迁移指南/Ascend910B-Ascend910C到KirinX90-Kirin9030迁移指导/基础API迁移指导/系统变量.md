# 系统变量

KirinX90/Kirin9030处理器不支持如下系统变量访问接口。

**表1** 系统变量兼容说明

| 基础API | 兼容说明 |
| :-- | :-- |
| CheckLocalMemoryIA | 
不支持。

CheckLocalMemoryIA监视设定范围内的UB读写行为，如果监视到有设定范围的读写行为则会出现EXCEPTION报错，未监视到设定范围的读写行为则不会报错。

该接口为调测接口，对功能无影响。
