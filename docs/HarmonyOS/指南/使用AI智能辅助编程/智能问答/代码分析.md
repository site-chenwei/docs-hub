# 代码分析

CodeGenie支持在**对话框中**输入对代码段和代码文件分析要求，帮助开发者快速理解代码逻辑、代码功能、技术细节和潜在问题等，提升开发效率。

DevEco Studio 6.0.2 Beta1之前版本，分析代码文件时需要通过**Files**入口选中文件；分析代码片段时，选中代码段后需点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/6a/v3/8uPlBfjDQTihlu3chujt0Q/zh-cn_image_0000002561832619.png?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=026ACD80599672C040720BA61802DB8FECF3706AAC16533FA2B9E4770537DFCF)图标开启光标上下文功能。

在DevEco Studio 6.0.2 Beta1版本，分析代码文件时，支持在对话框输入要分析的代码文件或直接分析当前文件；分析代码片段时，选中代码段后直接分析，无需开启![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a5/v3/tJBStyV0TxWxpjSgES1DVA/zh-cn_image_0000002561832615.png?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=7DE6BB5C8B2DD46901DACAD2B3977A5B9CFD980938C42A848A0F6CCBF83B21F4)图标。

从DevEco Studio 6.0.2 Release版本开始，使用HarmonyOS Ask智能体分析代码文件。

以DevEco Studio 6.1.0 Release和DevEco Studio 6.0.1 Beta1版本为例说明，操作如下。

**DevEco Studio 6.1.0 Release版本**

1.  选择HarmonyOS Ask智能体，在对话框中输入**@**符号选择**Files**，或点击**@****Add Context** > **Files**，选择需要分析的代码文件，或在对话框输入文件路径指定需要分析的代码文件，或分析当前代码文件。
2.  在对话框输入描述，点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/2a/v3/VLFyQ538RqGs8BQKiA75Dw/zh-cn_image_0000002561752635.png?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=BBE8009AFD91BB85C856AF1C8C6C82ADE2B85342E00ED69A6ADDF0B0C82B6964)发送后等待回复。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/20/v3/L6qBi3C3SiWxuEjb9zAojg/zh-cn_image_0000002530752696.gif?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=F842F667FEDCD83DE7234146A7A88B5B924EFBCD651AB806A2799997815C9EA6)

**DevEco Studio 6.0.1 Beta1版本**

分析代码文件：在对话框中输入**@**符号选择**Files**，或点击**@****Add Context** > **Files**，可对单个或多个代码文件进行分析。

分析代码片段：选中代码段，点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/f1/v3/THPmZHfPSaq1GVPsTJtY4A/zh-cn_image_0000002530912694.png?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=6F1745CC5CDF0013CD9C693AFE0C020384AC64C69C78B768CC3503333C1D80D1)图标开启光标上下文功能，在对话框内描述，开始分析。
