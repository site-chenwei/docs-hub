# 设计稿代码生成

从DevEco Studio 6.1.0 Beta1开始，CodeGenie支持上传Pixso设计网站导出的设计稿，基于设计稿生成应用/元服务可用的代码，生成结果支持实时预览，帮助开发者快速完成页面搭建。

#### 操作步骤

1.  点击页面右侧菜单栏CodeGenie图标完成登录后，可以通过如下三种方式进入上传设计稿入口：
    
    -   在对话框输入"/"调出命令，选择**Generate UI Code** **>****@Add Context** **> UX Design Draft**。
    -   在对话框左下角下拉框选择**Generate UI Code >@Add Context > UX Design Draft**。
    -   在模块右键选择**New > Page > Generate Pages with AI >@Add Context > UX Design Draft**。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e3/v3/fHpwedvjQGifr00Na45yfg/zh-cn_image_0000002561752645.png?HW-CC-KV=V1&HW-CC-Date=20260417T013645Z&HW-CC-Expire=86400&HW-CC-Sign=5B644CA0C8FDC9F48FD96717C59BB55EFD49DEF28B255D7513B738C18821F523)
    
2.  上传.pxhm后缀的设计稿文件，勾选解析出的设计稿，点击**Submit**提交，最多勾选和提交5张设计稿。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a8/v3/U82t9izcSFGhEz0r-VupfA/zh-cn_image_0000002530752706.png?HW-CC-KV=V1&HW-CC-Date=20260417T013645Z&HW-CC-Expire=86400&HW-CC-Sign=ED995355BE1A537AD617AD0DC746C65111EF3931F270E55CD85ED56EDEA5B130)
    
3.  对生成的页面进行预览。若提交的多张设计稿的底部导航栏相同，预览时支持点击导航栏进行切换；若提交的多张设计稿底部导航栏不同，默认预览第一张，不支持点击切换。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/92/v3/NhsCus1fSvG3-S3RasAzaw/zh-cn_image_0000002561832625.png?HW-CC-KV=V1&HW-CC-Date=20260417T013645Z&HW-CC-Expire=86400&HW-CC-Sign=446ED9494FDB502A92DC596EBD52EE4B39130A76BB8AE6432E4314920E57211D)
    
4.  点击**Save to Project**，在**Module**选择保存的模块和在**Page name**设置目录名称，点击**Next**。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/bb/v3/CcrDZ9KGR_yQMQKCPhonQg/zh-cn_image_0000002530752710.png?HW-CC-KV=V1&HW-CC-Date=20260417T013645Z&HW-CC-Expire=86400&HW-CC-Sign=14DF57B5C5BBA9CC439BB92276DEC726DE048A3377048862B14BA401E207ECC2)
    
5.  将生成的代码文件及资源保存至工程中，绿色文件为新增，蓝色文件为当前文档有修改，点击**Finish**完成添加。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c5/v3/0iD0t0t-SpqcGGH2SqzQjg/zh-cn_image_0000002530912704.png?HW-CC-KV=V1&HW-CC-Date=20260417T013645Z&HW-CC-Expire=86400&HW-CC-Sign=2F364245CC352C9E1CBD65730B0FA30980F25AD634AFB117F1AEC8AF530C6A50)
