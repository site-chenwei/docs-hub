# 自定义智能体配置

从DevEco Studio 6.0.1 Beta1开始，CodeGenie支持创建自定义智能体，增强AI问答能力，提升AI辅助编程和分析能力。自定义智能体创建包括模型上下文协议（MCP）配置、模型（Model）配置、自定义智能体（Agent）配置和调用。

从DevEco Studio 6.0.2 Beta1开始，自定义智能体新增规则（Rules）配置。

-   **[模型上下文协议（MCP）配置](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-agent-mcp)**  
    
-   **[模型（Model）配置](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-agent-model)**  
    
-   **[规则（Rules）配置](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-agent-rules)**  
    
-   **[自定义智能体（Agent）配置和调用](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-agent-use)**  
    
-   **[记忆（Memory）配置](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-memory)**  
    
-   **[自定义提示词库（Prompts）配置](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-prompts)**
