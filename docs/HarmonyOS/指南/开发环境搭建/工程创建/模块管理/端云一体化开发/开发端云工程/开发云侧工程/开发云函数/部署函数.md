# 部署函数

完成函数代码开发后，您可将函数部署到AGC云端，支持单个部署和批量部署。

单个部署仅部署选中的函数，批量部署则会将整个“cloudfunctions”目录下的所有函数同时部署到AGC云端。

下文以部署单个函数“my-cloud-function”为例，介绍如何部署函数。

1.  右击“my-cloud-function”函数目录，选择“Deploy 'my-cloud-function'”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/62/v3/5-qdXe8fQCWZ0JDZXxKLqg/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013643Z&HW-CC-Expire=86400&HW-CC-Sign=5A2CED70A9616FCFE46D436AF65FCE932B8FFBE4AFC7EA1511046322DF82675C)
    
    如需批量部署多个函数，右击“cloudfunctions”目录，选择“Deploy Cloud Functions”即可部署该目录下所有函数。如“cloudfunctions”目录下同时存在云函数和云对象，云函数和云对象将会被一起部署到AGC云端。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/30/v3/RTBr7vprTJ2R2B7Zy2kTug/zh-cn_image_0000002179498368.png?HW-CC-KV=V1&HW-CC-Date=20260417T013643Z&HW-CC-Expire=86400&HW-CC-Sign=418C547F545380A33CF26B8EE2E014A1207E6AF1152BD1AC7441AF0AE34EA1E8)
    
2.  您可在底部状态栏右侧查看函数打包与部署进度。
    
    请您耐心等待，直至出现“Deploy successfully”消息，表示当前函数已成功部署。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c6/v3/iVAbsi3jQ1aSPcCJRfL9Ig/zh-cn_image_0000002179498360.png?HW-CC-KV=V1&HW-CC-Date=20260417T013643Z&HW-CC-Expire=86400&HW-CC-Sign=D506C60F40698C5ACE59299A6468F1F97B17177833D940FD36CF9824C2361F44)
    
3.  在菜单栏选择“Tools > CloudDev”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/72/v3/ph0L6dxVR5-KbFcXxnCaSw/zh-cn_image_0000002214858997.png?HW-CC-KV=V1&HW-CC-Date=20260417T013643Z&HW-CC-Expire=86400&HW-CC-Sign=30C08A66021E16530BD3FEC714EE180492E9B61A2E236F4262E8DE924A8021FE)
    
4.  在打开的CloudDev面板中，点击“Serverless > Cloud Functions”下的“Go to console”，进入当前项目的云函数服务页面。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/75/v3/g8nDuPwdRpezDOUrUbDHmg/zh-cn_image_0000002214704617.png?HW-CC-KV=V1&HW-CC-Date=20260417T013643Z&HW-CC-Expire=86400&HW-CC-Sign=31C5F3B5854697A9236A493EAE6D68EA23E6D7AA045B32B34DEBB414A41671CB)
    
5.  查看到“my-cloud-function”函数已成功部署至AGC云端，函数名称与本地工程的函数目录名相同。
    
    部署成功后，您便可以从端侧调用云函数了，具体请参见[在端侧调用云函数](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/agc-harmonyos-clouddev-invokecloudfunc)。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/48/v3/_7FbhgeBSYuSZS24p2x91g/zh-cn_image_0000002179338680.png?HW-CC-KV=V1&HW-CC-Date=20260417T013643Z&HW-CC-Expire=86400&HW-CC-Sign=16B58A95CCD5FDB9D7D34418F1BBAE2E2FF016F53D300D287B7F292624911063)
