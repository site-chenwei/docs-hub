# 部署云侧工程

您也可选择在云函数和云数据库全部开发完成后，将整个云工程资源统一部署到AGC云端。

1.  右击云开发工程（“CloudProgram”），选择“Deploy Cloud Program”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8f/v3/nCaLUHMeSiCKM8vQ9uhnsQ/zh-cn_image_0000002214858797.png?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=756B70A42929C9E7E08624D402495C008A5A3B392E744F7A2219153278C3E4C4)
    
2.  您可在底部状态栏右侧查看云工程打包与部署进度。
    
    请您耐心等待，直至出现“Deploy successfully”消息，表示云工程已成功部署。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/d4/v3/lcQZDQEDTBiNZHNrNkrRAg/zh-cn_image_0000002179338472.png?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=402E8E6E766CBFE284708FB38FE0F7399C9DD3C00D21F86C814991961040FFEA)
    
3.  在菜单栏选择“Tools > CloudDev”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fd/v3/jrvnCMMpRXmMS-WRVGwpLw/zh-cn_image_0000002214858789.png?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=B9F38F986EED000CA8F5C16E3E93BEB34F544E22CB33A6B3F078AB22BD7D9099)
    
4.  在打开的CloudDev面板中，点击“Go to console”，打开当前项目的AGC云开发子控制台。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/6e/v3/0cqnA_gSRtSB9c-POji9fQ/zh-cn_image_0000002179498156.png?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=9E69DDAE79FCF0760B712E138AE35D3CB1A16DE7214CD8C55E38E9FF73DCCF14)
    
5.  分别进入云函数与云数据库服务菜单，可查看到您刚刚部署的云函数与云数据库资源。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/21/v3/RjWNPILwTrmgS-FNdQtRnw/zh-cn_image_0000002467759440.png?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=4F30E1271A28ED8052AF295E47665C7FB3490FF33F2A764A055315F6C4C54FC0)
