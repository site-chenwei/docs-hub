# （可选）通过CloudDev面板获取云开发资源支持

DevEco Studio为您提供了CloudDev云开发管理面板。该面板集成了AGC云开发子控制台、文档和社区入口，方便您直达AGC云开发子控制台进行服务和资源管理，并且可轻松跳转至各指导文档和社区论坛来获取技术支持，为您提供开发、调试、部署、管理与技术支持的端到端体验。

1.  在菜单栏选择“Tools > CloudDev”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/4b/v3/pvlpe7DATvaZVozhha-D7w/zh-cn_image_0000002179498248.png?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=76CE4BF1AE797BF64065B8FF4272F0992CEF8344FB8F578AFCD8343C67E2A977)
    
2.  在打开的云开发管理面板中，您可轻松获取各种云开发资源。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/59/v3/JWaA0uD1QPqtpF1nG0X-JQ/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=56D1691E463D27FAB3C5299BA8C48CD5885421D4085325E0B8D41B57997E4028)
    
    如尚未登录，请点击“Sign in”登录您的华为开发者账号。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ba/v3/9gv1ZROhSXWxoE6M0Y6DKA/zh-cn_image_0000002214704501.png?HW-CC-KV=V1&HW-CC-Date=20260417T013644Z&HW-CC-Expire=86400&HW-CC-Sign=CB22C02B03BA227063F7DE041BA2CCA9CA55A75FE983D37974E01F775FFCA04F)
