# 没有弹出用户授权界面

1.  首次使用运动健康App，请打开App同意相关的隐私声明。
    
2.  请确保已按照[请求用户授权](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/request_user_authorization)章节，完成功能开发。
