# 场景介绍

#### 手机与穿戴设备分布式协同

-   重要通知实现腕上提醒，实现即时通知推送，如：在手机侧App中设置日程提醒、用药提醒、任务提醒等数据，可以同步到穿戴设备侧App中，不用打开手机，也可以随时在穿戴设备上查看重要信息。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/28/v3/iF6c3xHBS_urdgaTtvBvWA/zh-cn_image_0000002568899135.png?HW-CC-KV=V1&HW-CC-Date=20260417T013559Z&HW-CC-Expire=86400&HW-CC-Sign=4AE90D96F1D26092F39B5CB0B9521C2B93A18C340A18FDCDC3839CD9CD1470DA)
    
-   为手机应用带来全新的“腕上交互”，如：用手机侧App看视频、听音频时，可以通过穿戴设备侧App操控手机侧App，比如暂停，下一首，停止等。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/d6/v3/2m2UrCjiS82qSMOBjfTHzA/zh-cn_image_0000002538019430.png?HW-CC-KV=V1&HW-CC-Date=20260417T013559Z&HW-CC-Expire=86400&HW-CC-Sign=5EAB2D2E8B1E1EA61F2F8421ECB204169E6110EF305015F14ECA51C70A1F4811)
    
-   手机侧和穿戴侧的实时协同体验。如：在手机侧App开启导航，可以通过穿戴设备侧App提示用户左转、直行、右转等。在走路或骑行时，不用每次拿出手机查看路线。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/69/v3/DRKHSnIIQnmNzi13-UPpuA/zh-cn_image_0000002538179358.png?HW-CC-KV=V1&HW-CC-Date=20260417T013559Z&HW-CC-Expire=86400&HW-CC-Sign=DD31469A124F16072A0624C0679087A9FB70FDDCE7F496638E9FCFBC386E512C)
    

#### 手机与穿戴设备虚拟化

开发者在手机侧开发App，而不需要在穿戴设备侧开发App。

-   穿戴设备状态实时掌握，如：手机侧App实时获取穿戴设备侧连接状态、佩戴状态、电量状态等，给用户提供更多增值服务。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/be/v3/rEO5I5mGS1ifQ0rI-r7k2g/zh-cn_image_0000002569019145.png?HW-CC-KV=V1&HW-CC-Date=20260417T013559Z&HW-CC-Expire=86400&HW-CC-Sign=CB5F0116F8803CB26764D9582326A8C0E78CEFC2F1A3178D049A829551C7B580)
    
-   穿戴设备传感器：手机侧App读取穿戴设备的传感器数据（如ECG、PPG、ACC数据等）。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ab/v3/ohJBqwELRgCh-NyMz0a2jA/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013559Z&HW-CC-Expire=86400&HW-CC-Sign=A5DDE36C9EF928D0B4E22E2D6463F2351A1872C382C7029608B6531E568B7E19)
    
    传感器受限发布，仅限专业研究机构使用。
