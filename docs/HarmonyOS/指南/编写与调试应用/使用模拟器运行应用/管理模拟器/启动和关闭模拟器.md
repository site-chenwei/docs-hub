# 启动和关闭模拟器

在设备管理器页面，单击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/28/v3/H7d59fyLQUWrUvz9-fb8-g/zh-cn_image_0000002530911082.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=FC45D3B7213CFEC4FC2098FEA31D1FF9F60F1746DAEFE5B9803AC781541CCFA8)即可启动模拟器。模拟器启动时会默认携带之前运行时的用户数据，包括用户上传的文件，安装的应用等。如果是新创建的模拟器，则不会携带用户数据。如果想清除之前运行时的用户数据，点击**Actions >** ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/18/v3/OAO9DzPwQJ-d-3D5wnpxXQ/zh-cn_image_0000002561831005.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=7292DC302A7231C250D4A223669D583DC92B7345AD14C14BE20B4500C1068D74) **> Wipe User Data**。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/60/v3/2fGjcL8ORaOibNjhkMc8Xw/zh-cn_image_0000002530751076.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=08764B1209E519DCC2D30A6A4B7C32FC7B0690DFDFABE45A6C5062A494E79F08 "点击放大")

从DevEco Studio 6.1.0 Beta1版本开始，如果创建模拟器时选择热启动，则启动模拟器时会加载上次关闭时保存的快照，启动后会恢复至上次关闭时的状态。热启动后，多屏状态会恢复为单屏状态，折叠屏模拟器会恢复成默认展开状态。

如果热启动后出现异常，可点击**Actions >** ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/71/v3/coGMIBSKSMqR04_AVf-wTg/zh-cn_image_0000002561751033.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=5D98594DD0128F8638822BB3A8BAFF43F79266164D95E6EAEA7A6DB857526567) **> Wipe User Data**清除用户数据后重新启动。

例如推包运行后关闭模拟器，再次启动时会显示在上次运行的界面。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/bf/v3/LFLntrvuSy-38iqTlm1Yjg/zh-cn_image_0000002530911092.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=FD1D0697BF45987B88CD2DD50C6130B09AC6D00F0A08EF843C6AE52C55D739BB "点击放大")

在模拟器运行期间，可以点击**Actions >** ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a2/v3/70NxIQEgTFa8tLh8BGjLjg/zh-cn_image_0000002561751025.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=FFA8A2B7DDA4CF2D80DD45577EC420310BD47624DF8878A80D4880880D1DDFA4) **> Show on Disk**显示模拟器在本地生成的用户数据。点击**Actions >** ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/de/v3/3NloMo-9T0uxHtj-PmuBwQ/zh-cn_image_0000002561751021.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=F976F22DF5648FC938EC5768EB12D44C9F0348528C6B3087E2786331382A66E2) **> Generate logs**可以生成模拟器自启动到此刻的所有日志信息。想要关闭运行中的模拟器，可以在设备管理器页面点击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/61/v3/U--4vh_jRG6NkXafhX23LA/zh-cn_image_0000002530751080.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=FB460BBF7F6BEFD6D7569EF09853215693497A382CB97E0F5CC97F8BD61AED6B)，或者点击模拟器工具栏上的关闭按钮![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/1b/v3/3h4mKttfS5a2cXiI3ET50g/zh-cn_image_0000002530751090.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=0DAFB9FB50F072B8B4BEBE50474325A2D5DCDA0AA58C08F31F473AEF241E2E64)。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c7/v3/5YVyPEu_RD6vrJMgXjzx4A/zh-cn_image_0000002561751013.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=D26240ACC028E075C8FCAC53CAF4D5F7C74084B3C1AEB33F9B3F5778A6A0E507 "点击放大")

模拟器关闭后，点击**Actions >** ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/9c/v3/VDq_-07RRc-N65zT7QQsvQ/zh-cn_image_0000002530911084.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=EB5310681AF02F55A58F7E03C55F78E20C68B08EBF8C0E4625E3CE428EC006CE) **> Delete**可以删除模拟器，并清除模拟器的用户数据和配置信息。
