# 创建模拟器

有网络环境可参考以下步骤创建模拟器，如果是无网络环境，请查看[离线部署模拟器](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-emulator-no-network)。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/27/v3/kt30QxiNQASajAFwTyr7lQ/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=228347D9D2951B4223ABD06C03F597F61B1BDD2CC1DF44D2630EBF99F628B52C)

在macOS中，您可能在活动监视器中发现模拟器进程占用的内存超过设置的内存。实际上，活动监视器中的**Memory**并不代表模拟器进程实际使用的物理内存，更多详情请参考[macOS上活动监视器中显示模拟器内存偏高](https://developer.huawei.com/consumer/cn/doc/harmonyos-faqs/faqs-app-running-23)。

#### 使用预置的模拟器

从DevEco Studio 6.1.0 Beta2版本开始，如果本地没有模拟器，DevEco Studio会预置模拟器，开发者无需创建即可快速使用。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/dc/v3/jtv6HydaSjuhjZMJ1X4NuA/zh-cn_image_0000002561831007.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=E268D9FE16999598F5BC93A33BE9F79FFCB5D2EC22C8F2C5A7AAFC7AF7D17F9A)

在设备选择框中，选择预置的模拟器并点击运行按钮![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/98/v3/5eB_tIu1QviOJBHkM7KUQQ/zh-cn_image_0000002561751019.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=310C41584DA5981F7D7B113D433A9717D5A5E3A6D08F03FDF3E56C1BB227279E)后，根据界面提示下载镜像，或点击菜单栏**Tools > Device Manager** >![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/18/v3/3JZKSf0tSGKWURd3L3txuw/zh-cn_image_0000002561751017.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=E586763BB937DDCDDA4C0718199CFC9A5D2CF6B35671E75FD4C310FCC6DAE081)下载镜像后，即可快捷使用模拟器。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c4/v3/NSbjBS7hTCaOFKqkkmv8Rg/zh-cn_image_0000002561831025.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=938A7959EFB8AFED0D74C7202A51BB6F243C7B05A3A2B642DC196550738FA1FF)

#### 创建新的模拟器

1.  点击菜单栏的**Tools > Device Manager**，点击右下角的**Edit**设置模拟器实例的存储路径**Local Emulator Location**，Mac默认存储在~/.Huawei/Emulator/deployed下，Windows默认存储在C:\\Users\\xxx\\AppData\\Local\\Huawei\\Emulator\\deployed下。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ed/v3/biBnh9pmRvy0M_KMHWnEJA/zh-cn_image_0000002530911088.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=FDB3878412707CB0A39C4FFCA3AFBE6E0D386BCE458FF4E6333963D951F98C1C "点击放大")
    
2.  在**Local Emulator**页签中，单击右下角的**New Emulator**按钮，创建一个模拟器。
    
    在模拟器配置界面，可以选择一个默认的设备模板，首次使用时请点击设备右侧的![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/1e/v3/tdq5cxcnSFWZtqmmAgl0DQ/zh-cn_image_0000002561831017.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=95E858C35AC1DFE0F9CB38B0BA950E9DC6671543CE550DFDD9B17C1CEA42BDC4)下载模拟器镜像，您也可以在该界面更新或删除不同设备的模拟器镜像。
    
    单击**Edit**可以设置镜像文件的存储路径。macOS默认存储在~/Library/Huawei/Sdk下，Windows默认存储在C:\\Users\\xxx\\AppData\\Local\\Huawei\\Sdk下。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/dd/v3/-70j64ypSG29FzwIEdNBgw/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=612DF4B2B9C91B0C39B9BE18C29EE6ED65286E3B9AE79D89C46E5C5084B8E349)
    
    如果配置界面显示异常，例如设备列表为空等，可先关闭DevEco Studio，并进入~/Library/Huawei（Windows路径为C:\\Users\\xxx\\AppData\\Local\\Huawei）目录，删除DevEcoStudiox.x文件夹（如DevEcoStudio6.0，具体文件夹名称和安装的DevEco Studio版本相关）以清理缓存。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ba/v3/MM6O9L_RRKK3CArza33uqQ/zh-cn_image_0000002561751027.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=13C1882E677363717BE52A662323D396371E5E1474E5BC39394D46330541CBDF)
    
3.  单击**Next**，设置设备相关的参数。从DevEco Studio 6.0.0 Beta1版本开始，部分设备支持自定义屏幕配置，具体支持的设备请参考[自定义屏幕配置](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-emulator-customize-screen-configuration)，下面以Phone为例。
    
    -   **Name**：设置模拟器的名称。
    -   **Screen Profile**：模拟器屏幕配置参数，可点击下拉框选择预置的机型配置，也可点击**Customize**自定义配置，在自定义配置的情况下可以对屏幕尺寸、分辨率和DPI进行修改，取值范围参考界面提示。
        -   **Screen size：**屏幕的对角线长度，单位为inch。
        -   **Resolution**：分辨率，包括宽度和高度，单位为px。
        -   **DPI**：像素密度，DPI 越高，UI组件占用的像素点越多，从而提供更精细的显示效果。
    -   **Boot options**：模拟器启动方式。从DevEco Studio 6.1.0 Beta1版本开始支持。
        -   **Cold boot**：以开机启动的方式重新启动。
        -   **Quick boot**：启动时加载上次关闭时保存的快照，启动后会恢复至上次关闭时的状态。
    -   **Memory**：设置模拟器的内存。
    -   **Storage**：设置模拟器的存储空间。
    
    确认所有参数后，点击**Finish**创建模拟器。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a1/v3/ql8ART36TMmwEs0YRbhQLQ/zh-cn_image_0000002530751096.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=803F5844B1A4ECB186488C9204A5439246860C1DC7C1C559142251B087385DCE)
    
4.  启动模拟器，有两种方式。
    
    -   从DevEco Studio 6.1.0 Beta2版本开始，创建后的模拟器会展示在设备列表中（最多10个），选择模拟器后，点击运行按钮![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8f/v3/WsgjT4jHQQegcrtRXIuT4g/zh-cn_image_0000002561831021.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=39C9116FB069586B644FEFA9369FD4C259FDE73E1928CEB564B77B9FD7C74C42)，即可一键完成启动模拟器、编译构建、推包运行操作。
        
        ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/aa/v3/p_JKWHBmS4W-RzEB90BNrQ/zh-cn_image_0000002561831023.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=88290C678C3922C7940FE7FCB13893D7CBC76B7060E19F5A9260F0D0DA801B77)
        
    -   在设备管理器页面，单击![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/4f/v3/McIaNKBlRd2QV983TW-Wig/zh-cn_image_0000002561751047.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=269FADE0E91D19D0090D0AEEA37CE6E65164A920AC89258B88FEC4B231E3E1F0)启动模拟器。
        
        ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/db/v3/ddtPT-WyQRm27n6hrLr9mA/zh-cn_image_0000002530751084.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=10777647F2E3EB27DB9B4B3BAC62270F5CD369CA949DD4FF9546899F1FF9EF8D "点击放大")
        
    
5.  单击DevEco Studio的**Run > Run'模块名称'**或![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/73/v3/RSljpfwHRt2atwZCRBieSw/zh-cn_image_0000002530911078.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=18FC614EA43B327B1F4B29F46C9EBD7E06D59E9A9B2BB0F115413327992AE5E1)。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/74/v3/8tjEQoBwRmmykT4_ExAJtg/zh-cn_image_0000002530751074.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=1426394900B27125B8E57BC15F61DCBA3E6AC1E7C25850050508D5C5A2BCBF11)
    
6.  DevEco Studio会启动应用/元服务的编译构建与推包，完成后应用/元服务即可运行在模拟器上。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fe/v3/GEx5WtHESWu5KlMgxxAHdg/zh-cn_image_0000002530911086.png?HW-CC-KV=V1&HW-CC-Date=20260417T013649Z&HW-CC-Expire=86400&HW-CC-Sign=0663E1F906EC4AC8A300260555D103C67C3B84A618EAA8781BD68F316EE95BD6)
