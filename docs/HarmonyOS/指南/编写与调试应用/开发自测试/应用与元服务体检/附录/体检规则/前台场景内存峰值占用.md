# 前台场景内存峰值占用

#### 规则详情

应用/元服务前台场景峰值内存占用：应用在前台且亮屏使用规程的内存占用应≤ 1500MB。

#### 检测逻辑

1.  执行hdc shell。
2.  执行hidumper --mem <进程pid>命令，获取如图Pss字段。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/f4/v3/eO7e5F_rTr-kPBV5451Okw/zh-cn_image_0000002561832611.png?HW-CC-KV=V1&HW-CC-Date=20260417T013651Z&HW-CC-Expire=86400&HW-CC-Sign=15D78CAAA932288BD5728518A2C2A0F88889FF7C3DF7C95027CAEA5A3BEA9CC4)

#### 计算逻辑

执行多轮测试，取最大Pss值为占用峰值，内存占用小于1500M。
