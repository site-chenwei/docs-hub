# 日志上传

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/12/v3/uoajYt33S8qx0WXDCByU-g/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013651Z&HW-CC-Expire=86400&HW-CC-Sign=0D070340E21B6776A2D2AD2E07CB68D915182D7471F1BB7E02EC11DC1D6A9253)

该功能仅支持中国境内（香港特别行政区、澳门特别行政区、中国台湾除外）。

若开发过程中遇到DevEco Studio卡顿、卡死或其他故障时，可点击IDE Error问题弹窗中Send Report，点击OK后向DevEco Studio回传日志信息。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ae/v3/GZjyPg8wQPyPPiCTBN-LVA/zh-cn_image_0000002561833669.png?HW-CC-KV=V1&HW-CC-Date=20260417T013651Z&HW-CC-Expire=86400&HW-CC-Sign=03649BE036B1C6ECC44BD80D16B610AFE73E08A79298500C9E4E82F68450B157)

或通过菜单栏**Help > Collect Logs and Diagnostic Data**，选择并上传相关日志，帮助DevEco Studio提升稳定性体验。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/84/v3/5s2HLzN0SCuXJnGnnPEacg/zh-cn_image_0000002530913742.png?HW-CC-KV=V1&HW-CC-Date=20260417T013651Z&HW-CC-Expire=86400&HW-CC-Sign=150C94F41FD94DE12ED979A7B479A6131E0BE75830FFB6C1D33B2BB40137F2C8 "点击放大")

若开发者后续需要关闭数据采集功能，请在**File > Settings**（macOS为**DevEco Studio > Preferences/Settings**）**\> Appearance & Behavior > System Settings > Data Sharing**设置界面，取消勾选**Send usage statistics**关闭数据采集开关。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/20/v3/pBXGIsUxR-icwT0xApkC0A/zh-cn_image_0000002530913746.png?HW-CC-KV=V1&HW-CC-Date=20260417T013651Z&HW-CC-Expire=86400&HW-CC-Sign=A3E74A02B7C3A10845584EE1E8F77EB3FF0BE693DBDFBA02EC03080DBEAFDE26)
