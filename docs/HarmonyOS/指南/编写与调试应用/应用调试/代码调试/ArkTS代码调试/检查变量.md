# 检查变量

当应用停止在某个断点处时，您可以在"Debugger"窗口中查看当前的变量信息。在"Frame"窗格中选择某个帧之后，可以在"Variable"窗格中检查变量，或对变量进行计算。

如需向"Watches"列表中添加变量或表达式，请按以下步骤操作：在"Watches"窗口中输入表达式，然后点击Add to Watches图标![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/38/v3/_IoAIYPBRD-GegcLSwGM8Q/zh-cn_image_0000002561832731.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=7F09BABB8E6F7F54E7A47D928F074D3F9E56673FA21C0CB4449028113F3C01E6)。

如需从"Watches"列表中移除某一项，点击鼠标右键，选择**Remove Watches**。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/97/v3/dWeFjM20TEKIYyAILXW0kQ/zh-cn_image_0000002561832733.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=056C3E794BD8E92FDDBAA21B6586B56DA13A5E46FB3F94E9ACE3959BF323353E)

从DevEco Studio 6.0.2 Beta1版本开始，支持预览pixelMap类型的变量。点击pixelMap变量右侧的**View PixelMap**，即可预览pixelMap。
