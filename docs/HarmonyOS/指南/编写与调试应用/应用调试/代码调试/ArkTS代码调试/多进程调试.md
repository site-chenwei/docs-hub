# 多进程调试

部分设备上，UIAbility支持以独立进程的方式运行并调试，详细请参考[进程模型](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/process-model-stage#其他进程类型)，可按照以下步骤对UIAbility进行调试。

#### 编译构建配置

1.  新建一个Ability，该Ability继承AbilityStage，作为独立进程的入口。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c/v3/lspIOWgDQsKWuaP9F5vP6Q/zh-cn_image_0000002561832779.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=07E3745738E3FDB0601F593BF2F4441BF6C521ED6328FD4E105C6253D91E3CED)
    
2.  右键ets目录，新建其它需要作为独立进程启动的UIAbility。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e/v3/1u89cRQvQX2B3PHG8iJamg/zh-cn_image_0000002530912852.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=8DC3A1103D10188E566F09B18B68B61AE5F4E35D125E799E8D703C2900A8CFB4 "点击放大")
    
3.  修改module.json5配置文件，增加独立进程入口及isolationProcess配置项。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/44/v3/-BfFULjgSG630OiBsEgvfw/zh-cn_image_0000002561752793.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=D15AF9A4271BA59BC073E36D4C4665BD7A666E76EE32D37A42B31982579F89D6)
    

#### 调试

1.  编写跳转UIAbility的代码。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fb/v3/sTXAuldSQKKjnafZ5xh6GA/zh-cn_image_0000002561832771.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=C477C9468BDE2EA7279DC85B50445C2AB1058E32B547260B4EFA53837F9C8DD7)
    
2.  在跳转的UIAbility中或独立进程入口处设置断点，启动调试。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/9a/v3/Wg5rgemtR3my-9vxP5MzLg/zh-cn_image_0000002561752791.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=0D330B5ACA2DDFF6F185E0315E641CC479A39B7FD9B6B7459867CB842DDE4304)
    
    跳转到以独立进程启动的UIAbility时将会新启动一个调试会话窗口。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ce/v3/v5x_S2aaTNCbA15XR5NKdA/zh-cn_image_0000002530752862.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=5D37B8A3E6D99989EC5E050746373BDC77701CFE9737AC5508B0501D38640A92)
