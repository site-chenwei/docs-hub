# 打印表达式

开发者可以通过Evaluate and log能力在代码执行到断点行时打印开发者指定的表达式。

1.  在需要打印表达式结果的地方设置断点。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/6d/v3/QvNfHGSDQAGn7R7MprTWNA/zh-cn_image_0000002561832883.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=2773862DBBD05C2AAD3201953CD83FB5D64119249E5BAA1FE31188136F83931B)
    
2.  右键断点，然后点击**More**按钮。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/38/v3/RgJduvTDR1K2P9a8m9Ejfg/zh-cn_image_0000002561752901.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=94B317A8F7B1CCD19F19DFD63AFFD5ED16D09B92E2B46048E26623B13F288F57)
    
3.  勾选**Evaluate and log**复选框，并在下方输入框输入要打印的表达式。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/4c/v3/z_TaFVliTHKGFL8vO3Nrfw/zh-cn_image_0000002530912956.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=A76839E7AF6FDA61FB0266EB54D66D867BB165DB515FD4157A7DA20711F1ADD1)
    
4.  启动调试，程序运行到断点时，切换到调试的Console窗口，表达式的打印结果将在这里展示。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/d7/v3/KR9ceRSfSxywUtzyzcPavA/zh-cn_image_0000002561832879.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=894ECEE4F94695A0F2E69525E81F07A4BB595882F7EE1EF7CB492C399B1B82E6)
