# 检查变量

调试时，在“Variables”页面查看变量，支持查看全局/静态变量、寄存器变量和局部变量。

#### 查看全局/静态变量

点击“Edit Configurations...”打开调试配置，在 native 调试配置界面中勾选“Show static/global variables in the Variables Pane”，调试过程中变量列表会展示全局/静态变量。

#### Simplify STL

在菜单栏点击“File > Settings（macOS为DevEco Studio > Preferences/Settings） > Build, Execution, Deployment > Debugger > C++ Debugger”，通过勾选“Display STL variables as visualization in the Variables Pane”在变量列表中展示简化后的 STL 变量值，或去掉勾选以展示其原始结构。

#### 变量监视

在"Watches"列表中输入表达式，然后点击Add to Watches 图标![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/89/v3/_Ck_gLcFSLKmn0aa7wXTQA/zh-cn_image_0000002561753711.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=8BF173D4281410EAA348A8B60315D21E881D6619CC2BC5D995B0AFFDEA00C636)，或在某个变量右键菜单中的“Add to Watches”添加监视的表达式，在每次程序停住之后会计算表达式的值。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/45/v3/dxBmFeMaRyCjgTGj0bqwtQ/zh-cn_image_0000002561833687.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=F0AF285E4DAD650DF39AE379269D8CDC7990E66196AD15AE759E187D5CD349D7)

#### 表达式求值

通过点击“Evaluate Expression...”按钮，或Watches 页面中的输入行中，输入表达式进行计算。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/71/v3/RqycjSCJR12bjEiLo66eBw/zh-cn_image_0000002530753770.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=391B2725E9FC6E7CB144675009EBCF2484B5B613753CEBDAA2D1CB80DC15AEFE)

#### 查看十六进制视图

在“Variables”页面点击鼠标右键，弹出框中选择“Show As Hex Values”，此时页面中的整型变量会以十六进制进行展示。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/4a/v3/EVxDziacSFGw4et13BU9yQ/zh-cn_image_0000002530753766.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=22BB9E174F3EB15A541DCFC228A344147121BD895C71969AFC509DBA6110103B)

#### 查看函数返回值

当使用“Step Out”从一个函数内步出后，变量列表中的“ReturnValues”会展示所步出函数的返回值。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c7/v3/L4F_ZwnFRkaUOhaF2Gd7sQ/zh-cn_image_0000002561753707.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=6F6CC071710093CB7AF62A95875939C4BAE269238FFDB658DB15254AC8F466B1)

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/61/v3/kJJ-c_tHSimTMZj0AbAC5Q/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=051A3063D1D4AD08402ECA82E689C75BA42C44040CBCCE789FD188ED75D4933D)

-   无法查看长度超过64位的数据结构。
-   无法查看引用类型返回值。
-   Step Out返回的位置存在断点时，无法查看函数返回值。

#### 其他说明

对于特定类型的变量，还支持查看bitmap预览、查看较长的字符串等功能。

-   ...View Bitmap：支持在调试时查看bitmap预览。

-   ...View：支持展开查看较长的字符串。
