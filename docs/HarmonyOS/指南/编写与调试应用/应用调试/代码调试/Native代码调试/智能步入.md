# 智能步入

进行C++调试时，当前代码行有多个函数调用时，开发者可以使用Smart Step Into功能![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/96/v3/UOD9RKN_SR6eB5uAMWztuA/zh-cn_image_0000002530752758.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=6A144604D2A998A5B0D06E20FC859398C76FE932D6C347D2D56D07CA8A586C1C)直接Step Into到其中某一个函数的实现中。

#### 操作步骤

通过点击调试窗口“entry-Native”调试器下的Debugger窗格中的按钮![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/53/v3/OPVHM05CRriV4W4RRGO5ag/zh-cn_image_0000002530912756.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=F47A8BCE1CF706AB514E7FDF1FE3AB4187DC5E089FC5895134EBCEEC275D8782)（或使用快捷键**Shift+F7**）触发Smart Step Into功能后，DevEco Studio会将当前代码中可以进行跳转的函数进行高亮显示。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/f0/v3/slbClOOSRiCCwVCSAa-Opw/zh-cn_image_0000002561752699.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=674B470CE4847B86711894035E81ED73E2FFF22031E51324878015D500BC8336 "点击放大")

开发者点击需要跳转的函数，程序会运行到目标函数的实现内。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/f5/v3/bshWWCN8TbCAaH2wvipH5Q/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=274CDB2B82EFCACB80A503D04E563C11B4D0D09A1E179A17C860CDB5753C980D)

已经执行完毕的函数不会高亮显示。
