# 使用调试器

Debug界面有三个tab页，分别是“entry”、“entry(PandaDebugger)”和“entry(Native)”。

通常第一个tab页“entry”用于展示推包安装过程。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/2f/v3/BDLExBbbTbm7oTA7VQZlMw/zh-cn_image_0000002530913488.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=D5DC26698CDF74C1A5786594010E5F16B2D786868CA10B9628A164AC0E6C39A7)

第二个tab页“entry(PandaDebugger)”和第三个tab页“entry(Native)”是调试器，用于调试Debugger功能，其中“entry(Native)”仅在涉及Native调试时才会拉起。调试器包含两个窗格，**[Debugger](#section1437520119316)**和**[Console](#section327153017314)**。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/59/v3/-9lnkyRySB2wsGaiSq63Lw/zh-cn_image_0000002530753478.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=90C9D22A4314BB2AE6EB421D8CD7133990EFDFC29E2B581B77512C6CB9BA46F2)

#### Debugger窗格

Debugger显示两个独立的窗格：

-   左侧区域是Frames，当应用调试到某个断点时，Frames区会显示当前代码所引用的代码位置。
-   右侧区域是Variables，用于展示当前变量。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/7a/v3/AI3CRD8QRjq5TcbeyWw4Yg/zh-cn_image_0000002561833399.png?HW-CC-KV=V1&HW-CC-Date=20260417T013650Z&HW-CC-Expire=86400&HW-CC-Sign=AC6F8D2CB5C36BE87D82B358A87064690B18CAF3E5B98A608115509709B3C3E2)
