# H5场景如何接入扫码

**问题现象**

Scan Kit没有提供H5的方案接入扫码。

**解决措施**

开发者可参考[前端页面调用应用侧函数](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/web-in-page-app-function-invoking)方式调用扫码接口，实现网页扫码功能。
