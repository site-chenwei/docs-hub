# 通过字节数组生成码图无法识别

**问题现象**

Scan Kit识别该码图内容显示为乱码，无法解析。

**解决措施**

通过字节数组生成码图，Scan Kit识别该码图内容显示为乱码，这种字节数组需要专门的解码器解析，例如地铁闸机。
