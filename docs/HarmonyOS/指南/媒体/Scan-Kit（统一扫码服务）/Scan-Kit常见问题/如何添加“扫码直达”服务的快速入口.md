# 如何添加“扫码直达”服务的快速入口

**问题现象**

用户编辑过控制中心，删除了默认存在的“扫一扫”入口，后续如何添加。

**解决措施**

控制中心编辑添加“扫一扫”入口：手机下滑菜单栏，打开控制中心，点击编辑按钮，在编辑区域中添加选择“扫一扫”后保存，即可在下拉控制列表中找到。
