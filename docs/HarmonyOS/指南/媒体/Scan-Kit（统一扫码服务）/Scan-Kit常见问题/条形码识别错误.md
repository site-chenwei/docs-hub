# 条形码识别错误

**问题现象**

条形码识别过程中存在码图类型识别错误的场景。

**解决措施**

条形码识别是通过粗细黑白条组成的模式来进行匹配，由于条形码的图案都比较相似，不同类型的码会出现误匹配现象，Scan Kit在不断优化扫码识别算法，持续提升识别的准确率。
