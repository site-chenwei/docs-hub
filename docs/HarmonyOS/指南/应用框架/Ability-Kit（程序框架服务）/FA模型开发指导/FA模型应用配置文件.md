# FA模型应用配置文件

应用配置文件中包含应用配置信息、应用组件信息、权限信息、开发者自定义信息等，这些信息在编译构建、分发和运行阶段分别提供给编译工具、应用市场和操作系统使用。

在基于[FA模型](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ability-terminology#fa模型)开发的应用项目代码下，均包含一个config.json配置文件，常用配置项请参见[应用/组件级配置](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/application-component-configuration-fa)。对于这两种配置文件的更多介绍，请参见[应用配置文件概述(FA模型)](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/application-configuration-file-overview-fa)。
