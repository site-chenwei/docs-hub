# 常见问题与异常处理

#### 应用数据迁移暂停

**问题现象1**

在数据加载界面，应用数据迁移暂停。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/a1/v3/_KKH5x7DRWK1MVRvg0utiQ/zh-cn_image_0000002569018915.png?HW-CC-KV=V1&HW-CC-Date=20260417T013544Z&HW-CC-Expire=86400&HW-CC-Sign=29C53E01A3154E01DA26E74FD661FFCED0A6F5C77CC46982BF6B6E076AAE7F38)

**可能原因**

应用数据迁移的过程中需要使用到网络，当前终端设备网络不可用，导致数据迁移暂停。

**解决方法**

单击“稍后连接WLAN加载”按钮，进入桌面后连接网络，终端设备网络可用后，恢复应用数据迁移。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/45/v3/mQAiNUmiRbuudAP0w8MmOw/zh-cn_image_0000002568898905.png?HW-CC-KV=V1&HW-CC-Date=20260417T013544Z&HW-CC-Expire=86400&HW-CC-Sign=2EA330408757682C1AB5CE6261530BFC3D6B3F7E73C302201C5A56213AB8AAEC)

**问题现象2**

进入桌面之后，若应用数据迁移还未结束，可通过通知栏进入应用加载界面查看加载进度

在应用加载界面，应用数据迁移暂停。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ac/v3/Z_9UqbRgSRKuZ7kkRuC3uA/zh-cn_image_0000002538019200.png?HW-CC-KV=V1&HW-CC-Date=20260417T013544Z&HW-CC-Expire=86400&HW-CC-Sign=38105FC68F9519753E46A483A07279A75508A813A039064DB7353D59CDB7E891)

**可能原因**

应用数据迁移的过程中需要使用到网络，当前终端设备网络不可用，导致数据迁移暂停。

**解决方法**

单击“稍后连接WLAN加载”按钮，进入桌面后连接网络，终端设备网络可用后，恢复应用数据迁移。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c4/v3/dMazwJ5qQJuotFqir_zUYg/zh-cn_image_0000002538179128.png?HW-CC-KV=V1&HW-CC-Date=20260417T013544Z&HW-CC-Expire=86400&HW-CC-Sign=3875659260A819E8AC675D6CD715518BC4C8FD7E6112A6129D9741DBC3DB3D26)

#### 应用数据迁移执行十五分钟后失败

**问题现象**

应用数据迁移执行十五分钟后显示失败。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/22/v3/NdYas_SzTWWlq2P986Numw/zh-cn_image_0000002538019198.png?HW-CC-KV=V1&HW-CC-Date=20260417T013544Z&HW-CC-Expire=86400&HW-CC-Sign=6D5E21ACF5297676629D0DE5C7A42C146794EB3FF379266FB2A4333AC4964D70)

**可能原因**

单个应用数据迁移执行超过十五分钟，超过设定的单个应用最长数据迁移时间，任务执行失败。

**解决方法**

请优化应用BackupExtensionAbility的代码实现，在十五分钟内完成应用数据迁移。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/fa/v3/5lO7NLu-Rde4Y--jHFywPg/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013544Z&HW-CC-Expire=86400&HW-CC-Sign=56C99E371498F120E6844FFB6C8BCB15A6CD5E8FF15DC67739708D595A3530D2)

已接入“数据迁移框架”的应用完成数据迁移后，才可以被消费者使用。尽可能快的完成应用数据迁移，可以带给消费者更好的体验。
