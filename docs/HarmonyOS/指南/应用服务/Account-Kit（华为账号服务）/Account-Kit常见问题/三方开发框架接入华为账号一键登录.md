# 三方开发框架接入华为账号一键登录

Account Kit随HarmonyOS整体框架支持三方平台，为常见的三方开发框架提供了SampleCode示例工程，体现了Account Kit的[华为账号一键登录](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/account-phone-unionid-login)特性，可参考示例工程进行应用的相关内容开发：

-   [Flutter](https://gitcode.com/HarmonyOS_Samples/accountkit-samplecode-clientdemo-flutter)
    
-   [H5](https://gitcode.com/HarmonyOS_Samples/accountkit-samplecode-clientdemo-h5)
    
-   [React-Native](https://gitcode.com/HarmonyOS_Samples/accountkit-samplecode-clientdemo-rn)
    
-   [uni-app](https://gitcode.com/HarmonyOS_Samples/accountkit-samplecode-clientdemo-uniapp)
