# 生态查询服务场景中获取自身场景值为空

**问题现象**

调用查询自身场景值接口返回场景值为空。

**解决措施**

当前应用/元服务没有生成场景值，通过其他应用或者元服务拉起当前应用或者元服务可以产生场景值，然后查询场景值。
