# 按需加载场景中，是否支持同时存在多个任务？

不支持，同一应用同时只能存在一个按需分发任务。

**解决措施**

1.  fetchModules接口支持多个模块同时下载安装，可以将多个任务合并为一个任务。
    
2.  通过on接口监听上一个任务的下载安装状态，安装成功后即可发起下一个任务。
