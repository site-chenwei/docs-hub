# 在AppGallery Connect中单独修改商品在某个地区的价格，以后汇率变动，是否会影响应用内的商品价格？

不会，除非开发者自己修改，修改方法请参考[修改数字商品](https://developer.huawei.com/consumer/cn/doc/app/revise-digital-products-0000001959074889)。
