# 调用应用归因服务登记归因来源接口报错身份检查错误

**问题现象**

媒体调用登记归因来源接口返回1009300003错误码，身份检查错误。

**解决措施**

1.  需要将应用通过应用市场上架。
    
2.  在应用归因云侧注册应用生态伙伴信息。
    
3.  开发者下配置应用id列表。
