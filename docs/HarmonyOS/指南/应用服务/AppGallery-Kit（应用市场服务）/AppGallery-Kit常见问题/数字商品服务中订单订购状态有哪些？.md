# 数字商品服务中订单订购状态有哪些？

1.  订阅型商品的订阅状态，请查看[订阅状态（status）](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/iap-query-subscription-status#subscriptionstatus)。
    
2.  消耗型/非消耗型商品，请查看[发货状态（finishStatus）和是否撤销的状态（purchaseOrderRevocationReasonCode）](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/iap-server-data-model#purchaseorderpayload)。
