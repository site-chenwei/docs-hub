# 概述

Screen Time Guard Kit需要经过用户授权才能对用户设备做时间管控和应用限制。Screen Time Guard Kit提供了请求用户授权和取消授权的接口，并且在健康使用设备的授权列表页面中，提供了用户主动打开/关闭授权的入口。开发者也可通过实现指定ExtensionAbility的回调方法，感知用户在健康使用设备的授权列表页中打开/关闭授权的行为。
