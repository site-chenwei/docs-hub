# 开通支付服务

请先参考“[应用开发准备](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/application-dev-overview)”完成基本准备工作及指纹配置，再继续进行以下开发活动。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/58/v3/QT05-Ur4RVeUgGoSOLD7Yw/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013620Z&HW-CC-Expire=86400&HW-CC-Sign=1D0BBBE4C88B2889E4E229A12A0F89B28DCE37696049092E87159F941DA9B170)

1.  后续接入涉及AppID绑定，限制只能企业开发者接入。开发者注册后发起实名认证请选择[企业开发者实名认证](https://developer.huawei.com/consumer/cn/doc/start/edrna-0000001062678489)，并且需要准备企业营业执照等必要材料。
    
2.  每个应用/元服务最多支持添加4个签名证书指纹。
    
3.  **[配置签名信息](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/application-dev-overview#section42841246144813)需选择手动签名**。
    

#### 开通步骤

1.  登录[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)网站，选择“开发与服务”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/07/v3/6ETNjvU-S-O73KMqoTv7aw/zh-cn_image_0000002568899725.png?HW-CC-KV=V1&HW-CC-Date=20260417T013620Z&HW-CC-Expire=86400&HW-CC-Sign=6309A34D4AE90C640369A39A1E0A99FC43E5B87619883E432F1D6AA62A8AE74B)
    
2.  在项目列表中找到项目（如未创建项目可点击添加项目先完成项目创建），在项目下的应用列表中选择需要开通Payment Kit的应用。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ea/v3/GI0EA3kNRWWERzfN_LL4Qw/zh-cn_image_0000002538020020.png?HW-CC-KV=V1&HW-CC-Date=20260417T013620Z&HW-CC-Expire=86400&HW-CC-Sign=DD67ADEE8AD1C10F6411C19D45EE5577BE5650B0CEAF784E5F295DE839B8895E)
    
3.  在左侧导航栏选择“盈利 > 鸿蒙支付服务（可在‘全部功能’中搜索服务并固定到导航栏） > 支付服务（非虚拟类）> 立即开通”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/7c/v3/huKml03ZSgSW81ISKn04ww/zh-cn_image_0000002538179948.png?HW-CC-KV=V1&HW-CC-Date=20260417T013620Z&HW-CC-Expire=86400&HW-CC-Sign=DDB876AAED7677DBBA3397F3A471CB3D26F2651B22DDB1D1F4754245398DC205)
    
4.  如涉及商户入网，在服务开通后需要为商户号申请绑定AppID，详细参见[商户号绑定AppID](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/payment-binding-appid-to-merc)（如未完成商户入网，可点击“申请支付商户号”先进行商户入网，详细介绍参考[商户入网](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/payment-merc-regist-apply)章节）。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/f8/v3/-NgV0mbhT6-jDZKcO0jSCQ/zh-cn_image_0000002569019735.png?HW-CC-KV=V1&HW-CC-Date=20260417T013620Z&HW-CC-Expire=86400&HW-CC-Sign=2071A7767103047249FB6474AC0C6EECCDDD52DC1BBEE6E013C5CD2CB48D688D)
