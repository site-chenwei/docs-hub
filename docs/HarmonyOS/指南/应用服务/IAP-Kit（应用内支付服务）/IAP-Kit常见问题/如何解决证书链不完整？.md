# 如何解决证书链不完整？

如果开发者提供的证书在IAP服务内置信任库中查询不到，则该证书不被IAP信任，需要构造完整的信任链以被IAP信任。

此处以Chrome浏览器最新版本（一般是支持自动验证证书链）为工具，以华为的证书为例，手工构造完整的证书链步骤如下：

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/84/v3/8b_k-2YASU-gdilqG871Gw/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013618Z&HW-CC-Expire=86400&HW-CC-Sign=8AE119EE126E715BB45DE85068FB17246D7D9F0896D2333EC997473FA9448B82)

开发者也可以选择其他证书链工具构造完整的证书链。

1.  查看服务器证书。
    
    访问[华为开发者网站](https://developer.huawei.com/consumer/cn/)，依次点击“查看网站信息 > 显示连接详情 > 显示证书 > 详细信息”，可查看证书状况，如下图所示：
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e2/v3/YB-N1uNHTOWAC6TXdKvcWA/zh-cn_image_0000002568899595.png?HW-CC-KV=V1&HW-CC-Date=20260417T013618Z&HW-CC-Expire=86400&HW-CC-Sign=FBD412DAD1E746450B317220579AFC45698B6D3D60CB070D4488AFE6C2A2746D)
    
2.  导出服务器证书链至文件中。
    
    依次点击“服务器证书 > 导出 > Base64 编码 ASCII，证书链（\*.pem;\*.crt） > 保存”，如下图所示：
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/92/v3/uImQMw-6TAyfy-F6jk0jsA/zh-cn_image_0000002538019890.png?HW-CC-KV=V1&HW-CC-Date=20260417T013618Z&HW-CC-Expire=86400&HW-CC-Sign=B42535D71C15251E267951A4FC761069C20A25C1D03C03030BA919062EE9991D)
    
3.  导出的证书链文件，使用文本编辑器打开.crt文件，可以看到与下图格式相似的PEM格式的证书内容，从上到下依次为“服务器证书 > 中间证书 > 根证书”，将已经拼接好的证书链返回给IAP服务器。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/97/v3/b-3NfaoEQXOv4mVjXYppcg/zh-cn_image_0000002538179818.png?HW-CC-KV=V1&HW-CC-Date=20260417T013618Z&HW-CC-Expire=86400&HW-CC-Sign=8EF2E1F93AFB3F47C5404FC2EE4458955196F09DB609759DF7CD0ADB1828F619)
