# 概述

#### 场景介绍

Share Kit支持手机和PC/2in1之间的碰一碰分享。利用PC/2in1设备的屏幕感知能力，识别手机轻碰屏幕的动作及位置，实现PC/2in1窗口级的交互。

#### 双向分享限制

从6.0.0(20) Beta5版本开始，手机与PC/2in1设备之间不支持双向分享。遵循以下机制：

-   当手机前台有可分享内容时，无论PC/2in1设备前台窗口是否有可分享内容，优先将手机作为发送端，PC/2in1设备作为接收端。
    
-   当手机前台无可分享内容且PC/2in1设备前台窗口有可分享内容时，PC/2in1设备作为发送端，手机作为接收端。
    
-   当手机前台和PC/2in1设备前台窗口均无可分享内容时，遵循无内容分享逻辑。
    

对于6.0.0(20) Beta3及之前的版本，当手机前台和PC/2in1设备前台窗口均有可分享内容时，支持双向分享（发送分享内容的同时也可接收到分享内容）。

#### 使用约束

-   手机与PC/2in1设备间碰一碰分享需登录相同的华为账号。
    
-   仅支持直板手机或折叠手机直板态与PC/2in1屏幕碰一碰分享。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/04/v3/dVRDFu7yQIyjmyyxo2kq_Q/zh-cn_image_0000002569019847.png?HW-CC-KV=V1&HW-CC-Date=20260417T013624Z&HW-CC-Expire=86400&HW-CC-Sign=903AC93DDF60D059D81396F185F42A10F0E3E9BA3377374910B63999FF355E26)
    
-   轻碰屏幕交互约束：
    
    -   手机与PC/2in1屏幕俯视夹角应≤5°。
        
        ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/13/v3/9orWc4wdTbKWOrEQ7L5sig/zh-cn_image_0000002568899839.png?HW-CC-KV=V1&HW-CC-Date=20260417T013624Z&HW-CC-Expire=86400&HW-CC-Sign=3EB6B9E162B43C9BB1338B9829872E66BD46B30EA247A92815D65CBE9EBAA8E9)
        
    -   手机与PC/2in1屏幕侧视夹角应＞35°。
        
        ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/03/v3/L2Gb6ty5TV-Pl-BhApFNrw/zh-cn_image_0000002538020134.png?HW-CC-KV=V1&HW-CC-Date=20260417T013624Z&HW-CC-Expire=86400&HW-CC-Sign=3D8669F025C0DCC7553085B6019D60CDF30DD59442AC232777E600F3084E2922)
        
    -   手机与PC/2in1屏幕正视夹角应≤25°。
        
        ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/2f/v3/GVYUlqV-RJqoT1nrW4lxYA/zh-cn_image_0000002538180062.png?HW-CC-KV=V1&HW-CC-Date=20260417T013624Z&HW-CC-Expire=86400&HW-CC-Sign=B2A5CD9057B7702CD9A89EC8EFBD37F334910BC30749480E9E634AF7F768CB19)

        ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/df/v3/XY0Kq_qCTJyHC0lavnCrlA/zh-cn_image_0000002569019849.png?HW-CC-KV=V1&HW-CC-Date=20260417T013624Z&HW-CC-Expire=86400&HW-CC-Sign=B12AED3F408ADA0A186F613258CFFCEBCB11E6245FB68684372E430BD2785CAD)
        
-   支持官方手机保护壳，不支持过厚的手机外壳。
    

#### 环境要求

-   支持的PC/2in1系统：[HarmonyOS 6.0.0 Beta1](https://developer.huawei.com/consumer/cn/doc/harmonyos-releases/overview-600#section1836613212578)及以上版本。
    
-   集成开发环境：[DevEco Studio 6.0.0 Beta1](https://developer.huawei.com/consumer/cn/doc/harmonyos-releases/overview-600#section1836613212578)及以上版本。
