# 剪贴板粘贴框遮挡智能填充选择框

**现象描述**

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/63/v3/UlfuGAeaS86Gw-rh_dRr1Q/zh-cn_image_0000002538020104.jpg?HW-CC-KV=V1&HW-CC-Date=20260417T013623Z&HW-CC-Expire=86400&HW-CC-Sign=8693AFBF0CF55AF89C6C25D181D2BFF38E4FA875F15B80B05F9FC3C7C0D0FEF9)

**解决措施**

在代码文件中设置.selectionMenuHidden(true)，使剪贴板粘贴框隐藏。

```typescript
      Row() {
        Text('收货人：').textAlign(TextAlign.End).width('25%')
        TextInput().width('75%').contentType(ContentType.PERSON_FULL_NAME).selectionMenuHidden(true)
      }
```
