# 申请优先通知权益

当用户终端收到携带[priorityNotificationType](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-notification-notificationrequest)字段的通知消息时，系统会将其识别为优先通知并优先显示。

申请优先通知权益存在以下限制：

1）该优先通知消息仅对商务办公、社交类应用开放。

2）应用仅在重要联系人、@我、加急消息、重要日程类提醒等场景下使用该能力。

#### 申请步骤

1.  登录[AppGallery Connect](https://developer.huawei.com/consumer/cn/service/josp/agc/index.html)，选择“开发与服务”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ae/v3/n5Y4WnJ8QEml3Hi-AAYuHg/zh-cn_image_0000002538179936.png?HW-CC-KV=V1&HW-CC-Date=20260417T013620Z&HW-CC-Expire=86400&HW-CC-Sign=F09B1F6E84C3389408E5CA57F9C3EB1B1C039DA5F1DCD79802C37F6D621AAEB3)
    
2.  在项目列表中找到您的项目，在项目下的应用列表中选择需要申请优先通知权益的应用。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c1/v3/HcK3KvOURmy6YT78TeF1zw/zh-cn_image_0000002569019723.png?HW-CC-KV=V1&HW-CC-Date=20260417T013620Z&HW-CC-Expire=86400&HW-CC-Sign=A5AC61408FFEFA6BB7BE9FD64D73E6BAD42D05D280D4491F8ABEA8F09C5010B5)
    
3.  进入“项目设置 > 开放能力管理”页面，点击“优先通知”的“申请”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/b1/v3/1NNszdDXSXGO7G9PG2fenA/zh-cn_image_0000002568899715.png?HW-CC-KV=V1&HW-CC-Date=20260417T013620Z&HW-CC-Expire=86400&HW-CC-Sign=10724A25CC488B8B16201E3BB41D3F7957C82D6A5661F19B0898BA15BD986818)
    
4.  开发者可参考“申请原因”中的模板，提供申请必须的相关信息，包括应用介绍、使用场景、申请用途、附件、承诺信息，然后点击“提交”按钮。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/92/v3/SMKjhVV-TXGheGlTEV5I7w/zh-cn_image_0000002538020010.png?HW-CC-KV=V1&HW-CC-Date=20260417T013620Z&HW-CC-Expire=86400&HW-CC-Sign=6B79F2C6D0BC00BC6D46D5BF20FAEF976139A2573109061A3A609554201E53EC)
    
5.  开发者可通过互动中心的“服务开通申请”消息获取优先通知权益申请结果。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/92/v3/C4V5KjkfRQi78V3vvMtI3A/zh-cn_image_0000002538179938.png?HW-CC-KV=V1&HW-CC-Date=20260417T013620Z&HW-CC-Expire=86400&HW-CC-Sign=835979CC36FCDEBECD772519B2FD174FBEB6598EC9F9D9F16FFB2851382978CC)

6.  优先通知权益申请通过后，须在“证书、APP ID和Profile”页面下左侧树形菜单的“Profile”页签，点击“添加”重新生成Profile文件，并下载Profile文件到本地，然后在“[发布应用](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-publish-app)”时，须将该Profile打包到应用包中。
