# 添加卡券时商户服务器主要需要执行哪些步骤？

1.  参考[钥匙开通](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/wallet-carkey-operation)推送模板数据至华为服务器。
    
2.  用户点击“按钮”或“链接”时将实例加密成JWE数据后推送给华为服务器。
