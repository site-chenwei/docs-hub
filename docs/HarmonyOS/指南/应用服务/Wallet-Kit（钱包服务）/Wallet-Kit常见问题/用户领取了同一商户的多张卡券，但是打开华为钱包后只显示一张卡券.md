# 用户领取了同一商户的多张卡券，但是打开华为钱包后只显示一张卡券

请检查[申请钥匙卡片](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/wallet-rest-api-carkey#申请钥匙卡片)的请求参数的serialNumber和organizationPassId字段是否都是唯一的。
