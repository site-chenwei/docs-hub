# 开发实况窗须知

当前实况窗场景请参见[实况窗支持对接的场景](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/liveview-introduction#实况窗支持对接的场景)。针对所有开放的场景，我们为开发者提供了实况窗[场景节点设计参考](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/liveview-design-formula#场景节点设计要求)，开发者可以根据参考设计场景内节点的数量与内容。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8d/v3/9UQtMomTQgaD3qTfg-dkRA/notice_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T013619Z&HW-CC-Expire=86400&HW-CC-Sign=6F787FB2FDD307E0BE9B500A0CB9C2C3D85BB1EA4367B1FA41E5CF67B5BFD183)

应用发送的实况窗需遵循《[实况窗设计规范](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/liveview-design-formula)》，不符合设计规范的方案将不被予以开通正式权限。同时若应用实况窗上线后出现违反实况窗设计规范的行为，将被视为[违规](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/liveview-introduction#违规处罚)。
