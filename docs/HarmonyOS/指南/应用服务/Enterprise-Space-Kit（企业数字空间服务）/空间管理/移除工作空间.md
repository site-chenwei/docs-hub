# 移除工作空间

#### 场景介绍

Enterprise Space Kit为企业用户提供移除工作空间的能力。

#### 接口说明

详细接口说明可参考[接口文档](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/enterprisespace-spacemanager#removeworkspace)。

| 接口名 | 描述 |
| :-- | :-- |
| [removeWorkspace](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/enterprisespace-spacemanager#removeworkspace)(localId: number): Promise<void> | 移除工作空间。使用Promise异步回调。 |

#### 开发步骤

1.  导入Enterprise Space Kit模块。
    
    ```typescript
    import { spaceManager } from '@kit.EnterpriseSpaceKit';
    ```
    
2.  调用[removeWorkspace](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/enterprisespace-spacemanager#removeworkspace)接口，移除工作空间，并且查看打印信息。
    
    ```typescript
    const workspaceId: number = 100;
    try {
      await spaceManager.removeWorkspace(workspaceId);
      console.info('Succeeded in removing workspace');
    } catch (err) {
      console.error(`Failed to remove workspace. Code: ${err.code}, message: ${err.message}`);
    }
    ```
