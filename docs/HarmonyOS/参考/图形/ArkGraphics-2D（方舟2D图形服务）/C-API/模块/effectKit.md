# effectKit

#### 概述

提供处理图像的一些基础能力，包括对当前图像的亮度调节、模糊化、灰度调节等。

**系统能力：** SystemCapability.Multimedia.Image.Core

**起始版本：** 12

| [effect\_types.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-effect-types-h) | 声明滤镜效果的数据类型。 |
