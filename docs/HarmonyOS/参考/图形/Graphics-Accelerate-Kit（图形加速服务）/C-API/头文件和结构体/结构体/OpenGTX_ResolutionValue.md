# OpenGTX\_ResolutionValue

#### 概述

此结构体描述游戏应用的分辨率值。

**起始版本：** 5.0.0(12)

**相关模块：** [GraphicsAccelerate](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/_graphics_accelerate)

**所在头文件：** [opengtx\_base.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/opengtx__base_8h)

#### 汇总

#### \[h2\]成员变量

| 名称 | 描述 |
| :-- | :-- |
| int32\_t [height](#height) | 分辨率高度值，以像素为单位，取值范围\[360,7680\]。 |
| int32\_t [width](#width) | 分辨率宽度值，以像素为单位，取值范围\[360,7680\]。 |

#### 结构体成员变量说明

int32_t OpenGTX_ResolutionValue::height
