# Camera\_MetadataOutput

```c
typedef struct Camera_MetadataOutput Camera_MetadataOutput
```

#### 概述

元数据输出对象。

可以使用[OH\_CameraManager\_CreateMetadataOutput](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-camera-manager-h#oh_cameramanager_createmetadataoutput)方法与OH\_CameraManager\_CreateMetadataOutputWithObjectTypes方法（从API version 23开始支持）创建指针。

**起始版本：** 11

**相关模块：** [OH\_Camera](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-oh-camera)

**所在头文件：** [metadata\_output.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-metadata-output-h)
