# Image

#### 概述

提供image接口的访问。

开发者可根据实际的开发需求，参考对应的开发指南及样例：

-   [图片解码](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/image-decoding-native)
-   [图片编码](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/image-encoding-native)
-   [图像变换](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/image-transformation-native)
-   [位图操作](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/image-pixelmap-operation-native)
-   [图片接收](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/image-receiver-native)

**起始版本：** 8

#### 文件汇总

| 名称 | 描述 |
| :-- | :-- |
| [image\_mdk.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-image-mdk-h) | 声明访问图像矩形、大小、格式和组件数据的函数。 |
| [image\_mdk\_common.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-image-mdk-common-h) | 声明图像常用的枚举值和结构体。 |
| [image\_packer\_mdk.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-image-packer-mdk-h) | 声明用于将图像编码到缓冲区或文件的api。可用于将像素数据编码到目标缓冲区或文件中。 |
| [image\_pixel\_map\_mdk.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-image-pixel-map-mdk-h) | 声明可以锁定并访问pixelmap数据的方法，声明解锁的方法。 |
| [image\_pixel\_map\_napi.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-image-pixel-map-napi-h) | 声明可以锁定并访问pixelmap数据的方法，声明解锁的方法。 |
| [image\_receiver\_mdk.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-image-receiver-mdk-h) | 声明从native层获取图片数据的方法。 |
| [image\_source\_mdk.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-image-source-mdk-h) | 声明将图片源解码成像素位图的方法。 |
