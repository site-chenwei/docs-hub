# OH\_AVSamplesBuffer

```c
typedef struct OH_AVSamplesBuffer OH_AVSamplesBuffer
```

#### 概述

LowPowerAVSink输入数据的结构体。应用在收到DataNeeded回调后需要将数据打包装进OH\_AVSamplesBuffer实例中送给对应的lowpower\_avsink。

**起始版本：** 20
