# AVScreenCapture

#### 概述

调用本模块下的接口，应用可以完成屏幕录制的功能。

开发者可根据实际的开发需求，参考对应的开发指南及样例：

-   [使用AVScreenCapture录屏取码流](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/avscreencapture-c-basic-process)
-   [使用AVScreenCapture录屏写文件](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/using-avscreencapture-for-file)

**起始版本：** 10
