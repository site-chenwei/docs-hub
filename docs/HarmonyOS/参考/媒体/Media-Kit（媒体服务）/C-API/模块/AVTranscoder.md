# AVTranscoder

#### 概述

提供请求转码能力的API。

根据实际的开发需求，参考对应的开发指南及样例[使用AVTranscoder实现视频转码](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/using-ndk-avtranscoder-for-transcodering)。

**系统能力：** SystemCapability.Multimedia.Media.AVTranscoder

**起始版本：** 20
