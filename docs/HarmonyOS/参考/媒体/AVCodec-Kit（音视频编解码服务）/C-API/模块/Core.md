# Core

#### 概述

Core模块提供用于媒体系统的基础骨干能力，包含内存、错误码、媒体数据结构等相关函数。

**系统能力：** SystemCapability.Multimedia.Media.Core

**起始版本：** 9

#### 文件汇总

| 名称 | 描述 |
| :-- | :-- |
| [media\_types.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-media-types-h) | 声明了常见媒体类型的定义。 |
| [native\_audio\_channel\_layout.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-audio-channel-layout-h) | 在录制和播放时的扬声器布局。 |
| [native\_avbuffer.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-avbuffer-h) | 声明了媒体数据结构AVBuffer的函数接口。 |
| [native\_avbuffer\_info.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-avbuffer-info-h) | 声明了媒体数据结构AVBuffer属性的定义。 |
| [native\_averrors.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-averrors-h) | 媒体框架错误码。 |
| [native\_avformat.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-avformat-h) | 声明了OH\_AVFormat相关的函数和枚举。 |
| [native\_avmemory.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-avmemory-h) | 声明了媒体数据结构AVMemory的定义。 |
