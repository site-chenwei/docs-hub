# OH\_AudioStreamBuilderStruct

```c
typedef struct OH_AudioStreamBuilderStruct OH_AudioStreamBuilder
```

#### 概述

声明音频流的构造器。构造器实例通常被用来设置音频流属性和创建音频流。
