# 模块描述

音频模板提供数据写入和接收功能，可以实现媒体应用接入音频模板控制方进行统一风格页面展示，并响应音频模板控制方的操作指令。

该模块提供如下功能：

[AVMusicTemplate](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/arkts-apis-avmusictemplate-avmusictemplate)：音频模板，可用于与音频模板控制方进行数据交互。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/bc/v3/bdTH5PrOQVizyoUQRSot1Q/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T014839Z&HW-CC-Expire=86400&HW-CC-Sign=29301B0E8876C1B40686086AA3BE54D3E47D817A3C4996F2AA1E8CCA980F43E7)

-   本模块首批接口从API version 23开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。
-   本模块仅适用于API version 23及以上版本的Car设备。

#### 导入模块

```ts
import { avMusicTemplate } from '@kit.AVSessionKit';
```
