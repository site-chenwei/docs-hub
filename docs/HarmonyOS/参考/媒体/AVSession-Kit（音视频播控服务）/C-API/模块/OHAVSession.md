# OHAVSession

#### 概述

提供播控模块C接口定义。

开发者可使用OHAVSession，作为媒体会话提供方接入AVSession服务，对应的开发指南及样例可参考[媒体会话提供方](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/using-ohavsession-developer)。

**系统能力：** SystemCapability.Multimedia.AVSession.Core

**起始版本：** 13

#### 文件汇总

| 名称 | 描述 |
| :-- | :-- |
| [native\_avmetadata.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-avmetadata-h) | 提供播控元数据的定义。 |
| [native\_avsession.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-avsession-h) | 媒体会话定义，可用于设置元数据、播放状态信息等操作。 |
| [native\_avsession\_errors.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-avsession-errors-h) | 提供播控错误码的定义。 |
