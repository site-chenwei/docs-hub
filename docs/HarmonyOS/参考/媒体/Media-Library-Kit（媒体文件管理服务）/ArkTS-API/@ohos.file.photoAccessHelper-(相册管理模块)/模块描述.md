# 模块描述

该模块提供相册管理能力，包括创建相册、访问和修改相册中的媒体数据。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/d3/v3/7I3FQBpyS6GK6QTsM6KV8w/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T014847Z&HW-CC-Expire=86400&HW-CC-Sign=249C7B73EA88D41FD3C64280B0818EEDBF934BA1FAE243B6081460FF0B033A8B)

本模块首批接口从API version 10开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

#### 导入模块

```ts
import { photoAccessHelper } from '@kit.MediaLibraryKit';
```
