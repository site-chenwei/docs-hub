# Vulkan

#### 概述

提供HarmonyOS平台扩展的Vulkan能力，扩展了使用OHNativeWindow创建Vulkan Surface的能力，以及获取OH\_NativeBuffer和OH\_NativeBuffer属性的能力。

**系统能力：** SystemCapability.Graphic.Vulkan

**起始版本：** 10
