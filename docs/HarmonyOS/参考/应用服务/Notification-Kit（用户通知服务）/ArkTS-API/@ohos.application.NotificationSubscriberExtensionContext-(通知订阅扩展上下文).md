# @ohos.application.NotificationSubscriberExtensionContext (通知订阅扩展上下文)

NotificationSubscriberExtensionContext模块提供通知订阅者扩展能力。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/40/v3/ToAG33mkQ0i5V9swQ1oQtA/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T014900Z&HW-CC-Expire=86400&HW-CC-Sign=4C937FCB9E9D2A4E9AC98B88088020E506BA5CB682B1EABE1EFA08FFFDFD2275)

本模块首批接口从API version 22开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

本模块接口仅可在Stage模型下使用。

#### 导入模块

```ts
import { NotificationSubscriberExtensionContext } from '@kit.NotificationKit';
```

#### NotificationSubscriberExtensionContext

NotificationSubscriberExtensionContext继承自[ExtensionContext](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-application-extensioncontext)类，是NotificationSubscriberExtensionAbility的上下文。

**系统能力**：SystemCapability.Notification.Notification
