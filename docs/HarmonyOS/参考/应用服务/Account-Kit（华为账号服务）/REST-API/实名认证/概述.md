# 概述

Account Kit提供了用户实名信息获取和实名信息校验能力，开发者可以通过Account Kit的相关功能，引导用户完成授权并校验其实名信息。

#### 场景介绍

-   **获取实名信息**
    
    应用服务器获取到用户授权后，调用华为账号服务器获取华为账号实名信息。
    
-   **实名信息校验**
    
    应用服务器获取到用户授权后，调用华为账号服务器校验用户实名信息。
