# memory

#### 概述

提供内存管理功能。提供的功能包括内存分配、内存释放等操作。

**起始版本：** 10

#### 文件汇总

| 名称 | 描述 |
| :-- | :-- |
| [purgeable\_memory.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-purgeable-memory-h) | 
提供可丢弃内存的内存管理功能。提供的功能包括创建、开始读取、结束读取、开始写入、结束写入、重建等。

使用时需要链接libpurgeable\_memory\_ndk.z.so。
