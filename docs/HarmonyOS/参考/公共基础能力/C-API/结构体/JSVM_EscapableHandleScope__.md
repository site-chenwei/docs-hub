# JSVM\_EscapableHandleScope\_\_\*

```c
typedef struct JSVM_EscapableHandleScope__* JSVM_EscapableHandleScope
```

#### 概述

表示一种特殊类型的handle scope，用于将在特定handle scope内创建的值返回到父作用域。
