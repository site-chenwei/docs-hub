# JSVM\_CallbackInfo\_\_\*

```c
typedef struct JSVM_CallbackInfo__* JSVM_CallbackInfo
```

#### 概述

表示传递给回调函数的不透明数据类型。可用于获取调用该函数的上下文的附加信息。

**起始版本：** 11

**相关模块：** [JSVM](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-jsvm)

**所在头文件：** [jsvm\_types.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-jsvm-types-h)
