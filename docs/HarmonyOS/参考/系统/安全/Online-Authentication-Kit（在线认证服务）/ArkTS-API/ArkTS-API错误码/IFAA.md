# IFAA

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/4b/v3/Dv-Hw6yiT5ubmOXu4T-6hA/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T014820Z&HW-CC-Expire=86400&HW-CC-Sign=AD4C589C5A003CAB60B84F85730EC67D81BF6FE8C8EFFF5099C46E3E477E535D)

以下仅介绍本模块特有错误码，通用错误码请参见[通用错误码](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/errorcode-universal)。

#### 1006100001 系统中断

**错误信息**

System Interruption.

**错误描述**

系统出现异常时，系统会产生此错误码。

**可能原因**

运行环境异常。

**处理步骤**

1 重启系统。

2 重试操作。

#### 1006100002 服务异常

**错误信息**

The service is abnormal.

**错误描述**

IFAA服务异常时，系统会产生此错误码。

**可能原因**

IFAA服务异常。

**处理步骤**

1 重试操作。

2 重启系统，重试操作。
