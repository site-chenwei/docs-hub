# BaseDdk

#### 概述

提供BASE DDK接口，包括创建共享内存，映射共享内存，取消映射共享内存，以及销毁共享内存。

**起始版本：** 12

#### 文件汇总

| 名称 | 描述 |
| :-- | :-- |
| [ddk\_api.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-ddk-api-h) | 声明主机侧访问的Base DDK接口。 |
| [ddk\_types.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-ddk-types-h) | 提供基础DDK接口所使用的Base DDK类型，宏定义，枚举值和数据结构。 |
