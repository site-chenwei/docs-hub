# 时间时区服务错误码

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c8/v3/tNbFXVu_S_WJehaltMCyQw/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T014831Z&HW-CC-Expire=86400&HW-CC-Sign=D716B9D5EF406640AD38166CE911597407011B200451E71C5519DA3C25715A06)

以下仅介绍本模块特有错误码，通用错误码请参考[通用错误码说明文档](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/errorcode-universal)。

#### \-1 时间时区服务异常

**错误信息**

Parameter check failed, permission denied, or system error.

**错误描述**

时间时区服务异常。

**可能原因**

系统运行异常。内存申请、多线程处理等内核通用错误。

**处理步骤**

系统运行异常。请确认内存是否充足。

#### 13000001 网络或操作系统异常

**错误信息**

Network connection error or OS error.

**错误描述**

网络或操作系统异常。

**可能原因**

网络或操作系统异常。网络无法连接或无法创建套接字等系统异常。

**处理步骤**

网络或操作系统异常。确认网络连接是否成功，系统资源是否足够。
