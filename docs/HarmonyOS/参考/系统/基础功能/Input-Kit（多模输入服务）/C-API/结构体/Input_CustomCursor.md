# Input\_CustomCursor

```c
typedef struct Input_CustomCursor Input_CustomCursor
```

#### 概述

自定义鼠标光标像素图资源。

**起始版本：** 22

**相关模块：** [input](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-input)

**所在头文件：** [oh\_input\_manager.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-oh-input-manager-h)

**相关接口：**

| 名称 | 描述 |
| :-- | :-- |
| [OH\_Input\_CustomCursor\_Create](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-oh-input-manager-h#oh_input_customcursor_create) | 创建自定义鼠标光标资源对象。通过调用[OH\_Input\_CustomCursor\_Destroy](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-oh-input-manager-h#oh_input_customcursor_destroy)销毁自定义鼠标光标资源对象。 |
| [OH\_Input\_CustomCursor\_Destroy](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-oh-input-manager-h#oh_input_customcursor_destroy) | 销毁自定义鼠标光标资源对象。 |
