# @ohos.app.form.FormEditExtensionAbility (FormEditExtensionAbility)

FormEditExtensionAbility模块提供卡片编辑功能，继承自[UIExtensionAbility](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-app-ability-uiextensionability)。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/c5/v3/H3_BhPmpT1KLrFnq1GPg7A/note_3.0-zh-cn.png?HW-CC-KV=V1&HW-CC-Date=20260417T014815Z&HW-CC-Expire=86400&HW-CC-Sign=0BFE22AA462213B710746D69B79EC9F4A8FEBE969EBEE63CCFB22FE1E5D5FE10)

本模块首批接口从API version 18开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

本模块接口仅可在Stage模型下使用。

#### 导入模块

```ts
import { FormEditExtensionAbility } from '@kit.FormKit';
```

#### FormEditExtensionAbility

提供卡片编辑功能。

#### \[h2\]属性

**模型约束：** 此接口仅可在Stage模型下使用。

**系统能力：** SystemCapability.Ability.Form

| 名称 | 类型 | 只读 | 可选 | 说明 |
| :-- | :-- | :-- | :-- | :-- |
| context | [FormEditExtensionContext](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inner-application-formeditextensioncontext) | 否 | 否 | FormEditExtensionAbility的上下文环境，默认值是继承自UIExtensionContext的对象。 |
