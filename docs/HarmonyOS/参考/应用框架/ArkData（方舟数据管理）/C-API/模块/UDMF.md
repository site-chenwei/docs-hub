# UDMF

#### 概述

统一数据管理框架旨在定义数据跨应用、跨设备以及跨平台过程中的各项标准，提供统一的HarmonyOS数据语言和标准化的数据接入与读取通路。

**起始版本：** 12

#### 文件汇总

| 名称 | 描述 |
| :-- | :-- |
| [udmf.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-udmf-h) | 提供访问统一数据管理框架数据的接口、数据结构、枚举类型。引用文件：<database/udmf/udmf.h> |
| [udmf\_err\_code.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-udmf-err-code-h) | 声明统一数据管理框架错误码信息。引用文件：<database/udmf/udmf\_err\_code.h> |
| [udmf\_meta.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-udmf-meta-h) | 声明统一类型数据信息。引用文件：<database/udmf/udmf\_meta.h> |
| [uds.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-uds-h) | 提供标准化数据结构相关接口函数、结构体定义。引用文件：<database/udmf/uds.h> |
| [utd.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-utd-h) | 提供标准化数据类型描述相关接口和数据结构。引用文件：<database/udmf/utd.h> |
