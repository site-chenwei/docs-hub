# OH\_Rdb\_ConfigV2

```c
typedef struct OH_Rdb_ConfigV2 OH_Rdb_ConfigV2
```

#### 概述

管理关系数据库配置，与[OH\_Rdb\_Config](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-rdb-oh-rdb-config)的区别是该结构体成员变量不对外暴露，使用一系列方法配置该结构体的属性，支持向量数据库。

**起始版本：** 14

**相关模块：** [RDB](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-rdb)

**所在头文件：** [relational\_store.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-relational-store-h)
