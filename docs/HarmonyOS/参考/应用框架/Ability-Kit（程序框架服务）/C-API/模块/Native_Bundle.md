# Native\_Bundle

#### 概述

提供查询应用包信息的功能，包括应用包名、应用指纹、应用appId等。

**起始版本：** 9

| 名称 | 描述 |

| [ability\_resource\_info.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-ability-resource-info-h) | 提供组件资源信息的接口，用于获取组件的以下信息：包名、模块名、组件名、图标、分身索引、是否为默认应用等。 |
| [bundle\_manager\_common.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-bundle-manager-common-h) | 声明BundleManager定义的相关错误码。 |
