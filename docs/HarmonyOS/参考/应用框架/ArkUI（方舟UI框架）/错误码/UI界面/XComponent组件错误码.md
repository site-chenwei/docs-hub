#### 103501 XComponent状态异常

**错误信息**

XComponent is in invalid state.

**错误描述**

当前XComponent状态异常，方法调用失败。

**可能原因**

参考报错接口的说明。

**处理步骤**

确保XComponent持有的Surface状态正常。
