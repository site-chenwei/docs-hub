# ArkUI\_AccessibilityEventInfo

```c
typedef struct ArkUI_AccessibilityEventInfo ArkUI_AccessibilityEventInfo
```

#### 概述

无障碍事件信息。当无障碍服务或辅助应用要求控件执行操作后，需要发送执行成功事件。
