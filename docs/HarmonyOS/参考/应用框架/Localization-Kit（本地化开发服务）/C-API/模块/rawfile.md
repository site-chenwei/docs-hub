# rawfile

#### 概述

提供操作rawfile目录和rawfile文件的功能，包括遍历、打开、搜索、读取和关闭rawfile。

**起始版本：** 8

#### 文件汇总

| 名称 | 描述 |
| :-- | :-- |
| [raw\_dir.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-raw-dir-h) | 提供rawfile目录相关功能。 |
| [raw\_file.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-raw-file-h) | 提供rawfile文件相关功能，功能包括搜索、读取和关闭。 |
| [raw\_file\_manager.h](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-raw-file-manager-h) | 提供资源管理rawfile相关功能，可以使用ResourceManager打开rawfile进行后续相关操作，像搜索和读取等。 |
