# 连接设备，hdc shell命令可以查看到设备连接状态，但是DevEco Studio连接显示会闪跳一下，显示无设备

DevEco Studio工具无法识别设备，参考常见问题[设备无法识别](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/hdc#设备无法识别)解决，或执行hdc kill -r命令重启hdc后重新尝试连接。
