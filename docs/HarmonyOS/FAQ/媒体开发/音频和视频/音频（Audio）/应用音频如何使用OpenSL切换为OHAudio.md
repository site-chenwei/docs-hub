# 应用音频如何使用OpenSL切换为OHAudio

由于OpenSL ES无法满足音频系统的能力拓展，开发者可通过使用OHAudio替代OpenSL ES开发音频业务，具体请参考链接：[从OpenSL ES切换到OHAudio](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/replace-opensles-by-ohaudio)。
