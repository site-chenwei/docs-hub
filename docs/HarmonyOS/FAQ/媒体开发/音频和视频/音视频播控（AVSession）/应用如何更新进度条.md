# 应用如何更新进度条

如果应用希望在播控中心支持进度显示和控制，需要将资源的时长信息设置给AVSession，并注册seek的回调接口以响应系统的进度控制。应用可以在倍速或播放状态发生变化时更新进度条，以节约系统资源。
