# 应用内如何进行输出设备的选择

系统未直接提供切换输出设备的接口，但支持通过投播组件实现设备切换。集成AVCastPicker到应用界面，即可实现设备切换。AVCastPicker还支持自定义能力，以满足应用界面一致性的需求。
