# 如何在播控中心显示歌词

创建AVSession后，设置AVMetadata的歌词内容。歌词格式必须为标准的lyric格式，其他格式不支持。

[滚动歌词](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/basic-playback-control#滚动歌词)
