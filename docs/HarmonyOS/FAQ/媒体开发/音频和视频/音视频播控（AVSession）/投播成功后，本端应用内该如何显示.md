# 投播成功后，本端应用内该如何显示

应用在感知到设备连接成功后，客户端界面应适配为遥控器状态，以提升投播体验。

具体可以参考[开发步骤](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/distributed-playback-guide#开发步骤)中的第四步的描述。
