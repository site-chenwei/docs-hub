# 如何自定义投播组件

投播组件支持自定义能力。

支持图标默认颜色、选中颜色、深浅色适配及组件内元素自定义。

实现步骤，可以参考链接：[自定义样式实现](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/using-switch-call-devices#自定义样式实现)。
