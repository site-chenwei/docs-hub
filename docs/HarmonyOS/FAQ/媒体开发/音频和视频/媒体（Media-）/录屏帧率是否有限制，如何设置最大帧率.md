# 录屏帧率是否有限制，如何设置最大帧率

可以调用OH\_AVScreenCapture\_SetMaxVideoFrameRate()设置录屏时的最大帧率，实际帧率受限于设备能力，具体规格可参考[设置录屏时的最大帧率](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-native-avscreen-capture-h#oh_avscreencapture_setmaxvideoframerate)。录屏的帧率需同时满足编解码的规格，请参考[设置正确的视频帧率](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/obtain-supported-codecs#设置正确的视频帧率)。
