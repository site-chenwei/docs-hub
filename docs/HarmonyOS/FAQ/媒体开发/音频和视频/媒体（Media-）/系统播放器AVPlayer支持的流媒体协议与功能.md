# 系统播放器AVPlayer支持的流媒体协议与功能

系统播放器 AVPlayer 支持 HLS、HTTP 和 HTTPS 协议，具备播放、暂停和 Seek 功能（Seek 功能不支持精准定位）。切集和全屏功能需由用户在应用层实现。
