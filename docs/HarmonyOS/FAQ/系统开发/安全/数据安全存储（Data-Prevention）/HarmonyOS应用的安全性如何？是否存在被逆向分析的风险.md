# HarmonyOS应用的安全性如何？是否存在被逆向分析的风险

HarmonyOS的反编译安全性提供基础的应用加固安全能力，包括混淆、加密和代码签名，保护代码免受反编译和反调试。高级混淆功能目前由第三方支持。
