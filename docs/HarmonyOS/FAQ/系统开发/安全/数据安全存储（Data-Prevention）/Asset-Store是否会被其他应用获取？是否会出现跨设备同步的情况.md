# Asset Store是否会被其他应用获取？是否会出现跨设备同步的情况

1\. Asset Store关键资产存储服务当前未实现应用共享数据功能，仅写入数据的应用可以访问。

2\. 如果开发者不希望跨设备同步数据，在添加关键资产时需将SyncType属性设置为never。
