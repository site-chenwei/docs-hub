# 如何保存私钥等重要数据

可以通过密钥导入接口将私钥导入密钥管理服务，导入后在安全环境（如TEE）中使用，提升安全性。此外，也可以使用关键资产存储服务加密保护，详情请参考[Asset Store Kit（关键资产存储服务）](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/asset-store-kit)。
