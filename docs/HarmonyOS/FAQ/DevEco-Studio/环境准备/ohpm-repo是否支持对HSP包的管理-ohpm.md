# ohpm-repo是否支持对HSP包的管理

目前ohpm-repo支持.har和.tgz两种包管理。HSP仅支持应用内HSP，应用间HSP尚不支持。

在使用ohpm-repo时，可以参考[三方中心仓](https://ohpm.openharmony.cn/#/cn/home)。
