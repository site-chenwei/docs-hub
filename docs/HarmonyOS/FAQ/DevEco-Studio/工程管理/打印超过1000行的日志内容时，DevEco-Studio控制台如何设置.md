# 打印超过1000行的日志内容时，DevEco Studio控制台如何设置

1.  导出日志：Help->Show Log in Explorer。
2.  在DevEco控制台中更改日志输出设置：依次选择File > Settings > Editor > General > Console，然后勾选Override console cycle buffer size选项，自定义缓冲区大小。
