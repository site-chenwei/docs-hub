# 打开工程时左侧目录树不显示

**问题现象**

当删除工程目录下的.idea/modules文件夹或该文件夹不存在时，如下图所示。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/5b/v3/InWP19TuQ4CZUMFUUwu8sA/zh-cn_image_0000002229758589.png?HW-CC-KV=V1&HW-CC-Date=20260417T020320Z&HW-CC-Expire=86400&HW-CC-Sign=3EDB2E106653BF3CE6F034E2BA4675F85ED1D1DC988D6032F8BD01AC4DDD0110)

重新打开工程，左侧目录树未显示，如下图所示。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/01/v3/H7PQXFiqSlyqSy_Ry5mO4g/zh-cn_image_0000002194318328.png?HW-CC-KV=V1&HW-CC-Date=20260417T020320Z&HW-CC-Expire=86400&HW-CC-Sign=75B077B6507F737AA79BDEF6BAC68A6FE5D7D0A68F7305D38E3780AB11CD9F80)

**问题原因**

modules文件夹下的iml文件定义工程模块结构信息，modules.xml定义模块结构文件的位置。删除modules文件夹后，无法找到对应的iml文件。

**解决措施**

关闭工程，删除文件管理器中的modules.xml文件，然后通过DevEco Studio重新打开工程，工程目录树即可恢复。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/93/v3/MZCFZf-ARCCw0YRPb5gAMw/zh-cn_image_0000002194158720.png?HW-CC-KV=V1&HW-CC-Date=20260417T020320Z&HW-CC-Expire=86400&HW-CC-Sign=203E0B8D7592F16C46C679F33F4EB9D6A546A6BC7DBD6C5834A73207DC4D97E4)
