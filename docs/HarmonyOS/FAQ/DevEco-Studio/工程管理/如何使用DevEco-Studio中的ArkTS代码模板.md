# 如何使用DevEco Studio中的ArkTS代码模板

使用代码模板可以提高开发效率。在DevEco Studio中，可以通过“File \\> Settings \\> Editor \\> Live Templates”查看现有模板并新增自定义模板。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/2f/v3/k1avcJOCQNyD4IqLvFNs5A/zh-cn_image_0000002194318448.png?HW-CC-KV=V1&HW-CC-Date=20260417T020320Z&HW-CC-Expire=86400&HW-CC-Sign=71F77B46CA8312DCCA705B8FB27B4FDEA907A83ABB7EF5EED4ADB93FF44ED7B9 "点击放大")

点击展开按钮查看现有模板详情，点击右上角加号新增自定义模板。

1.  使用现有代码模板可以提高开发效率。

2.  自定义代码模板。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/01/v3/E0HRaTn2R7-BQ88rBvbBKQ/zh-cn_image_0000002229604221.png?HW-CC-KV=V1&HW-CC-Date=20260417T020320Z&HW-CC-Expire=86400&HW-CC-Sign=9F2DE6A15EA4CDC7AE3415D58CC75A0DE3F4CC0335F40D2605DA9296A29E25F2 "点击放大")![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/79/v3/OXV0MKUkS-Ku1HVuQPenGg/zh-cn_image_0000002194158848.png?HW-CC-KV=V1&HW-CC-Date=20260417T020320Z&HW-CC-Expire=86400&HW-CC-Sign=260771C29B8F45F8C0019B665635EFCFC1D7415055F23CE524095857612ED017 "点击放大")
    
    选择“Template Group”，新增模板群组以表示自定义模板。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8b/v3/nfi5UNetRV6hggiJnCCnIQ/zh-cn_image_0000002229604229.png?HW-CC-KV=V1&HW-CC-Date=20260417T020320Z&HW-CC-Expire=86400&HW-CC-Sign=D7FF01B59EE13CFF8C511FDC9E0CB5C33A522377E41EC0B7A2937D81AD70EA6B "点击放大")
    
    选中自定义群组，点击+，新增模板。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/18/v3/5qtOva8CR9-QQPTcZ6qIsQ/zh-cn_image_0000002194318456.png?HW-CC-KV=V1&HW-CC-Date=20260417T020320Z&HW-CC-Expire=86400&HW-CC-Sign=7CFEA9C02368661CE19389598FB8F7F14A7CE29F7E39A89D3171C4CEAEC7F68D "点击放大")
    
    以定义Component模板为例，在Abbreviation中输入模板名称，此名称作为代码模板的快捷键。在Template text中编辑模板样式，其中$符号标记用户需自行输入的部分。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/cd/v3/2huWJLK3QqueeFrLQVoXew/zh-cn_image_0000002229604225.png?HW-CC-KV=V1&HW-CC-Date=20260417T020320Z&HW-CC-Expire=86400&HW-CC-Sign=311D2EEC859121C9E7DF0AFAFF73FDB08EA0216D5CB04A01DCCB86D21F944D16 "点击放大")
    
    点击上图的“Change”处，选择代码模板文件。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8c/v3/1yJyYzM0S42FB96qbQW7lg/zh-cn_image_0000002194158856.png?HW-CC-KV=V1&HW-CC-Date=20260417T020320Z&HW-CC-Expire=86400&HW-CC-Sign=FFE566A512D611ACAD24A22487F66C2A5F10B2AAAC98A3F76FCDE8BF20F76A15 "点击放大")
    
    点击Edit variables，在弹窗中设置LABEL值的自动获取方式。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/57/v3/dkKih3LATtifHCjqHvdAYA/zh-cn_image_0000002229604237.png?HW-CC-KV=V1&HW-CC-Date=20260417T020320Z&HW-CC-Expire=86400&HW-CC-Sign=A765F2E777BF9502276D879323CFD2234C128759A34A756C8851F2874DA74611)
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/f4/v3/jBTX5EaJQT6p6P53iEx6QA/zh-cn_image_0000002229758725.png?HW-CC-KV=V1&HW-CC-Date=20260417T020320Z&HW-CC-Expire=86400&HW-CC-Sign=1D50C0451FEB16F92652984E8DCB1237819EED1864A3CDAB845C51EF85F44D18)
    
    自定义代码模板生效。
