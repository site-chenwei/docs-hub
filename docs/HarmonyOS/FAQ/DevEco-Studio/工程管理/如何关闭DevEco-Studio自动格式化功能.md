# 如何关闭DevEco Studio自动格式化功能

如果是复制C/C++的代码到DevEco Studio中会自动格式化，可以在File -> settings打开设置界面，搜索Code Style，找到C/C++，选择Other，去掉Format On Paste的勾选。
