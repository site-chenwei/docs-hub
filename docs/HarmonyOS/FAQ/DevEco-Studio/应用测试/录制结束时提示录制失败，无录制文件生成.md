# 录制结束时提示录制失败，无录制文件生成

**可能原因**

录制能力依赖的uitest检测能力被其他程序占用。

**解决措施**

重启手机或使用命令以杀死该服务，命令如下：

```screen
hdc shell killall -9 uitest
```
