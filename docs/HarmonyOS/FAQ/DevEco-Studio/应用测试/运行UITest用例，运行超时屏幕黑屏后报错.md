# 运行UITest用例，运行超时屏幕黑屏后报错

**问题现象**

运行UITest用例时，如果屏幕黑屏并超时，将报错提示“Tests failed”。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/77/v3/aWVeI0DTSQSampjwISld4Q/zh-cn_image_0000002229758661.png?HW-CC-KV=V1&HW-CC-Date=20260417T020326Z&HW-CC-Expire=86400&HW-CC-Sign=14C7FCBC9741ECA218095D21A2236C818659A8740F3A41F56D75D3DB14305EC2)

**解决措施**

运行UITest用例时，需确保设备屏幕常亮。屏幕常亮时，用例能够成功运行。
