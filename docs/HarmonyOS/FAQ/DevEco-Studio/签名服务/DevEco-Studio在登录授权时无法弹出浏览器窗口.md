# DevEco Studio在登录授权时无法弹出浏览器窗口

**问题现象**

在DevEco Studio中点击登录、帮助文档时，无法弹出浏览器窗口。

**解决措施**

可以参考如下操作换一个浏览器：

1.  可以在“Settings > Tools > Web Browsers and Preview”中找到浏览器设置。
2.  点击图中红框的按钮将Edge置于顶层，也就是把默认浏览器设置为Edge等电脑内已经存在的浏览器，即可解决该问题。
