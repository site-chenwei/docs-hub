# 如何解决编译报错“Error: the name 'XXX' can only contain \[a-zA-Z0-9\_\]”的问题

**问题现象**

编译报错：文件名只能包含大小写字母、数字、下划线。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/cd/v3/bmDihwxQRMu_8ISaWzKOWA/zh-cn_image_0000002194158636.png?HW-CC-KV=V1&HW-CC-Date=20260417T020323Z&HW-CC-Expire=86400&HW-CC-Sign=89EC7A85AA5DF22087D547B29B36ADAD10E002A147FD6A62ABD04A0270859A86)

**解决方案**

检查文件名是否包含除了大小写字母、数字和下划线之外的其他字符。
