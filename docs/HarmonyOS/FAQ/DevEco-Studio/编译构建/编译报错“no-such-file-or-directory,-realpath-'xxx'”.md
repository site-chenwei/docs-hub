# 编译报错“no such file or directory, realpath 'xxx'”

**问题现象**

DevEco Studio编译时出现错误，提示“no such file or directory, realpath 'xxx'”错误信息。![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/2e/v3/lpsIl7kXTs6NkSUJ54hNdg/zh-cn_image_0000002229758637.png?HW-CC-KV=V1&HW-CC-Date=20260417T020322Z&HW-CC-Expire=86400&HW-CC-Sign=192490DD604F0A62C449AA29E111A8F9EA222D0A75FA05423DE4AD9FEE799FE3)

**解决措施**

“no such file or directory”是一种常见的错误提示，表示当前工程无法找到指定文件或目录。该错误可能由以下原因引起：

1.  检查报错路径是否真实存在。
2.  检查文件或目录路径的正确性，包括文件名、目录名和字母大小写。
3.  检查权限：如果文件或目录存在，确保工程有足够权限访问。
