# HarmonyOS NDK上的llvm是否有更改，一些中间产物能否复用

**问题描述**

HarmonyOS NDK上的llvm是否有更改，一些中间产物能否复用？

**解决方案**

HarmonyOS NDK上的llvm仅支持ohos编译的中间产物复用，不支持非ohos编译的产物复用。
