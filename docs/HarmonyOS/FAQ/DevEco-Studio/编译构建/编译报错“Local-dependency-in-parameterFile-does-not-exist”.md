# 编译报错“Local dependency in parameterFile does not exist”

**错误描述**

parameterFile 中的本地依赖项不存在。

**可能原因**

parameterFile中定义的值不存在。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/9c/v3/MZMAgMqeSpm-BEAak4H3jw/zh-cn_image_0000002194318024.png?HW-CC-KV=V1&HW-CC-Date=20260417T020323Z&HW-CC-Expire=86400&HW-CC-Sign=D5DE7B2097A9B316CDB094987A7458420C039DC022D249EC6B2309897E2F0ADE)

**解决措施**

检查本地依赖路径是否存在，以及是否区分大小写。
