# 生成签名时报错删除 .p12 文件目录下的 material 文件夹，重新应用自动签名

**问题描述**

点击生成签名时出现错误：

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/28/v3/Fey3SNgMRteSCnc67ZyPag/zh-cn_image_0000002229604377.png?HW-CC-KV=V1&HW-CC-Date=20260417T020322Z&HW-CC-Expire=86400&HW-CC-Sign=DED41E1636185D53736855DFCEEFDCE5E624095778882BBA2C69434EAA1F8E35)**解决方案：**

可以通过签名界面提供的profile文件（\*.p7b）或Certpath文件（\*.cer）对应的签名文件路径，删除本地的material文件夹，然后重新启动DevEco Studio进行签名。
