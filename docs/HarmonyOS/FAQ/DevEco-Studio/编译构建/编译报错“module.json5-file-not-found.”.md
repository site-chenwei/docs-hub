# 编译报错“module.json5 file not found.”

**错误描述**

未找到module.json5文件。

**可能原因**

模块下的module.json5文件缺失。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/60/v3/3_p8v3b4QVuRLwolhxELkg/zh-cn_image_0000002229604093.png?HW-CC-KV=V1&HW-CC-Date=20260417T020323Z&HW-CC-Expire=86400&HW-CC-Sign=B59DA430403D448739EFAB3F6312591C36BBD14B3E56AA80545CFD7D6961E01F)

**解决措施**

在Stage模型项目中，确保每个模块包含module.json5文件。
