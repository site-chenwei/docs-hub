# 模拟器上部分视频无法正常播放

**问题现象**

在模拟器播放视频时，可能会遇到视频有声音但无画面，或者播放卡顿的问题。

**解决措施**

由于模拟器仅支持RGBA格式的像素显示，请确保视频解码格式正确。模拟器仅支持软件解码，如果视频解码器依赖硬件解码，视频可能无法正常播放。
