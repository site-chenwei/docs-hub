# 设备管理获取模板数据提示网络异常，下载模拟器镜像提示网络异常

**问题现象**

-   **场景一**：设备管理获取模板数据失败，错误提示：“Network request failed. Verify your network connection and Emulator is available in your country/region.”
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/ec/v3/B2uOvFlQSqmPE9ZfEJrBJA/zh-cn_image_0000002194318324.png?HW-CC-KV=V1&HW-CC-Date=20260417T020324Z&HW-CC-Expire=86400&HW-CC-Sign=4EDEC74BB3DEB84D3ADAF7AA0BB2873BD8433DB93B4EA4088E0F379E222B0B79)
    
-   **场景二**：模拟器镜像下载失败，提示“The network or server is abnormal.”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/27/v3/klVIgk9dSli0kylhrFQRRw/zh-cn_image_0000002229758585.png?HW-CC-KV=V1&HW-CC-Date=20260417T020324Z&HW-CC-Expire=86400&HW-CC-Sign=63C2CE2CB876160EEFE1FD52154078D1F5D2CF2FB860ACA577A7122760827C0A)
    
-   **场景三**：打开设备管理，界面显示为空，错误提示：“Network request failed. Verify your network connection and Emulator is available in your country/region.”
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/e7/v3/yo9g2OJFQ3-eDR-CP4trGQ/zh-cn_image_0000002313594524.png?HW-CC-KV=V1&HW-CC-Date=20260417T020324Z&HW-CC-Expire=86400&HW-CC-Sign=B9A8DE5D53BD581C52454EE6FDBB099CEA6692C12B07790BF387631EDE126A52)
    

**解决措施**

1.  尝试清除本机DevEco Studio缓存文件后重启，缓存目录：
    
    Windows:C:\\Users\\xxx\\AppData\\Local\\Huawei\\DevEcoStudioX.X\\caches
    
    Mac：~/Library/Caches/Huawei/DevEcoStudioX.X/caches
    
2.  尝试修改本机网络环境后进行重试，例如：[配置Proxy代理](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-environment-config#section10369436568)、连接手机热点、关闭VPN。
3.  请检测您的网络并确认您当前电脑环境或华为账号是否在[模拟器支持的国家/地区](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-emulator-devicetype)内。
