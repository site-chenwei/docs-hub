# Windows X86模拟器启动后无法亮屏

**问题现象**

更新DevEco Studio后，首次启动Windows X86模拟器，模拟器无法亮屏。

**解决措施**

清除模拟器中的用户数据后重新启动。具体操作请参考[启动和关闭模拟器](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-emulator-start-and-close)。
