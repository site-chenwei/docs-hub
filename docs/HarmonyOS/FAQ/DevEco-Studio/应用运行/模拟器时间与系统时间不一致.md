# 模拟器时间与系统时间不一致

**问题现象**

模拟器运行一段时间后，显示的时间可能与系统当前时间不同步。

**解决措施**

在设置中开启自动时间同步，联网时时间将自动更新，也支持手动设置时间。重启模拟器后，时间会自动同步。
