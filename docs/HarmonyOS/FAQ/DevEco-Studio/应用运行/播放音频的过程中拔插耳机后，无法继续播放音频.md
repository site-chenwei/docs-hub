# 播放音频的过程中拔插耳机后，无法继续播放音频

**问题现象**

拔掉USB接口的耳机，再重新插上后，模拟器的音频播放功能将失效。

**解决措施**

模拟器不支持耳机的热插拔。请勿在播放音频时插拔耳机。如果插拔耳机后无法播放音频，请重启模拟器。建议使用圆孔接口耳机以避免此问题。
