# 编辑器自动联想、跳转等功能失效

**问题现象**

DevEco Studio上代码自动联想、跳转等功能失效。

**解决措施**

可能是启用了省电模式，需要将其关闭。开发者可通过取消File > Power Save Mode选项的勾选以关闭省电模式。取消勾选后如下图所示，表示省电模式已关闭。
