# 设备ROM升级至5.0.0.31及以上版本，DevEco Studio连接设备不成功，查询不到设备

**问题现象**

当设备ROM升级至5.0.0.31及以上时，DevEco Studio连接设备失败，无法查询到设备。

**解决措施**

确认开发环境所在电脑的用户名是否为中文。如果用户名是中文，请将其修改为英文或拼音，然后重新尝试连接设备。
