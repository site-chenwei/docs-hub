# 如何强制重启设备

可以通过以下几种方法重启设备。

-   方式1：长按电源键，然后点击重启。
-   方式2：长按音量减键和电源键。
-   方式3：在设备已连接数据线且可被识别的情况下，使用以下命令：hdc shell reboot
