# 如何查看项目中har版本冲突

属于ohpm install --all的信息，可以执行此命令查看并保留对应信息。

[ohpm list](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-ohpm-list)可以查看已安装的三方库；依赖冲突解决可以使用overrides字段控制版本，也可参考[模块内依赖版本冲突](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/ide-ohpmrc#section1623415477574)文档。
