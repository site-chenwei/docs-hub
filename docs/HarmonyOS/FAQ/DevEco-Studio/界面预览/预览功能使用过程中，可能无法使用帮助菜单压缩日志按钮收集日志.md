# 预览功能使用过程中，可能无法使用帮助菜单压缩日志按钮收集日志

**问题现象**

当同时预览多个工程时，点击帮助菜单中的“压缩日志”按钮，可能会因日志文件被占用而无法压缩。

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/f1/v3/tecwHa8hS7WPt9HJ1hU48A/zh-cn_image_0000002229603945.png?HW-CC-KV=V1&HW-CC-Date=20260417T020321Z&HW-CC-Expire=86400&HW-CC-Sign=CF96D38AC89716316A485428D9609DFC0D59E582C6988460B5FF665B690F91FD)

此时右下角会出现压缩失败的提示：

![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/1d/v3/C1ab7_UCQLOzCJXTpGQJ2Q/zh-cn_image_0000002229758417.png?HW-CC-KV=V1&HW-CC-Date=20260417T020321Z&HW-CC-Expire=86400&HW-CC-Sign=564D6BB7EF51F6300816A2C31DC303AAB7D2365C474D513279CFB73C2AB851A6)

**解决措施**

请关闭预览过的工程，或者重启DevEco Studio后不要打开预览器，即可再次压缩收集日志。
