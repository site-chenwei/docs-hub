# 系统弹框和应用消息推送的区别是什么

-   系统弹框是操作系统在特定情况下弹出的提示框，例如系统更新提示、电量低提示。应用程序无法主动触发或控制。
-   应用消息推送是指应用程序在后台发送的消息通知，显示在手机的通知栏或电脑的任务栏。例如，社交媒体的新消息提醒、邮件应用的邮件到达提醒。用户可以根据需要设置是否接收这些通知。
