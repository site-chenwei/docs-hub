# 如何获取组件的属性信息

可通过通用属性-组件标识内的[getInspectorByKey](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-attributes-component-id#getinspectorbykey9)方法获取组件所有属性信息。
