# 如何设置自定义弹窗位置

初始化自定义弹窗时，通过alignment参数设置弹窗对齐方式，通过offset参数设置弹窗偏移量。

[弹窗的选项（包含alignment、offset等参数）](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-promptaction#basedialogoptions11)
