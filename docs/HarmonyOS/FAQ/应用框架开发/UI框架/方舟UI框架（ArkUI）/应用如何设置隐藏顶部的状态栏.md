# 应用如何设置隐藏顶部的状态栏

在UIAbility的onWindowStageCreate生命周期中，设置setWindowSystemBarEnable接口。

onWindowStageCreate(windowStage: window.WindowStage): void {
  windowStage.getMainWindowSync().setWindowSystemBarEnable(\[\])
  // ...
}
