# 如何获取屏幕的宽度、高度、分辨率和横竖屏等信息

引入屏幕属性模块，通过调用[display.getDefaultDisplaySync()](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-display#displaygetdefaultdisplaysync9)方法获取display对象后，可获取到屏幕的宽度、高度、分辨率和横竖屏等信息。
