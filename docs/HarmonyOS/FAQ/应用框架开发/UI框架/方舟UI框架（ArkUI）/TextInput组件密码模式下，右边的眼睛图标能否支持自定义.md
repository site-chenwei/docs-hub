# TextInput组件密码模式下，右边的眼睛图标能否支持自定义

支持，可使用TextInput的[passwordIcon](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-basic-components-textinput#passwordicon10)设置输入框末尾的密码可见性切换图标。
