# 如何根据组件内容大小修改浮动窗口

使用组件的onAreaChange方法监听区域变化，根据返回的内容大小修改浮动窗口大小。

[组件区域变化事件](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/ts-universal-component-area-change-event)
