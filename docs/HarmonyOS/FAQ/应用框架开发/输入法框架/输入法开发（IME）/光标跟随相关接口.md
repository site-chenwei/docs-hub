# 光标跟随相关接口

在输入法应用中，可以使用InputMethodEngine(输入法服务)模块的[keyboardDelegate.on('cursorContextChange')](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/js-apis-inputmethodengine#oncursorcontextchange)接口，订阅光标变化事件。
