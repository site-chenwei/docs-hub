# 如何在ArkTS中使用闭包

**问题描述**

ArkTS是否支持TypeScript中的闭包函数？

**解决措施**

ArkTS不支持在函数内声明函数，可以使用lambda函数作为替代。

**参考链接**

[不支持在函数内声明函数](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/typescript-to-arkts-migration-guide#不支持在函数内声明函数)
