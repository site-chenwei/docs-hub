# 如何使用内置的js引擎？JIT支持策略如何

使用内置的JS引擎可以参考：[JSVM](https://developer.huawei.com/consumer/cn/doc/harmonyos-references/capi-jsvm)，推荐使用自带的JS引擎，不禁止使用第三方引擎，但禁止使用第三方引擎的JIT功能。
