# 如何快速生成class的setter和getter方法

1.  在类内部，右键点击，选择“生成”。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/8c/v3/9vAVIJ7rQk-zxGSKDeOKNA/zh-cn_image_0000002229758765.png?HW-CC-KV=V1&HW-CC-Date=20260417T020302Z&HW-CC-Expire=86400&HW-CC-Sign=95AB8C27E7EC5C44C791023E831EE596FD7FC23FD68D00792ACB1C0B5895E543 "点击放大")
    
2.  选择Getter and Setter。
    
    ![](https://contentcenter-vali-drcn.dbankcdn.cn/pvt_2/DeveloperAlliance_scene_100_1/93/v3/2TwtlEoTTPiwpbMYPUvRjw/zh-cn_image_0000002229604269.png?HW-CC-KV=V1&HW-CC-Date=20260417T020302Z&HW-CC-Expire=86400&HW-CC-Sign=975203ADB8D98402F03FD3E2FF226C69C44B4804FF7525D19BFB51D8BEF99E8A "点击放大")
