# 如何将数据库数据内置到App

在用户使用前，可以将预置数据从其他格式导入关系型数据库中。预置数据可以存储在本地资源文件或自定义的模块类组件中。如果选择使用文件保存数据，建议将其存储在应用的内部目录中。

数据库持久化操作，参考文档：[通过用户首选项实现数据持久化](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/data-persistence-by-preferences)

应用数据持久化，是指应用将内存中的数据通过文件或数据库的形式保存到设备上；参考文档：[应用数据持久化概述](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/app-data-persistence-overview)

关系型数据库的持久化，参考文档：[通过关系型数据库实现数据持久化](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/data-persistence-by-rdb-store)
