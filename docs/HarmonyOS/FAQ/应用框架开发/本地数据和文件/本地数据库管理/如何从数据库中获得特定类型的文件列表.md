# 如何从数据库中获得特定类型的文件列表

DataSharePredicates是开发者通过DataShare框架查询数据库中的数据所使用的筛选条件。它经常被应用在更新数据、删除数据和查询数据中。谓词的接口函数与数据库的筛选条件一一对应。
