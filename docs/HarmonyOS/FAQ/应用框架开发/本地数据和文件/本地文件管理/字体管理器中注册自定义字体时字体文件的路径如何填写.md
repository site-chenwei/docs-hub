# 字体管理器中注册自定义字体时字体文件的路径如何填写

建议将字体文件放置在rawfile目录中，通过路径/data/storage/el1/bundle/entry/resources/rawfile/进行访问。请注意，放置文件会占用应用安装所需的存储空间，避免放置过大的文件。
