# HAR、HSP不能支持Ability、Page声明，限制的理由是什么？后续是否会支持

Page：HAR和HSP支持page；HAR中的page需要通过命名路由的方式跳转，具体请参见参考资料。Ability：HAR和HSP不支持，推荐在HAP中配置Ability。
