# 使用发布证书进行调试时出现安装错误: Install Failed: error: failed to install bundle.

应用市场发布的证书无法通过非应用市场渠道安装。 若要测试发布证书应用，可以参考以下2种方式：

-   API ≤ 9 的应用请使用[开放式测试](https://developer.huawei.com/consumer/cn/doc/AppGallery-connect-Guides/agc-betatest-introduction-0000001071477284)。
-   API ≥ 10的应用请使用[邀请测试和公开测试](https://developer.huawei.com/consumer/cn/doc/app/agc-help-harmonyos-testapp-0000001873653977)。
