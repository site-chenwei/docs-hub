# 如何获取resource目录下的资源

可以使用以下两种方法获取资源文件：

1\. 使用 $r 或 $rawfile 获取静态资源。程序运行时，资源路径不会改变。

2\. 使用ResourceManager获取资源。适合动态获取，运行时可动态更改资源路径。
