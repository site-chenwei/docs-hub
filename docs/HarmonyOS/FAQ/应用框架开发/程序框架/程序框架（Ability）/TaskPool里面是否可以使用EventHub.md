# TaskPool里面是否可以使用EventHub

EventHub仅支持在主线程中使用，不支持在TaskPool中使用。

[使用EventHub进行数据通信](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/uiability-data-sync-with-ui#使用eventhub进行数据通信)
