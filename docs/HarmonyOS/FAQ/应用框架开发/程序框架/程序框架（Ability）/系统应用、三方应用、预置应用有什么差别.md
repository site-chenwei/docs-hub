# 系统应用、三方应用、预置应用有什么差别

-   系统应用：profile证书中app-feature为hos\_system\_app的应用，系统应用能调用系统接口。
-   三方应用：profile证书中app-feature为hos\_normal\_app的应用，三方应用不能调用系统接口。
-   预置应用：厂商在出厂时预先安装在设备上的应用，可以是系统应用，也可以是三方应用。
