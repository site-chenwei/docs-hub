# 如何在UIAbility、页面和组件中获取UIAbilityContext

可以在UIAbility中或者页面中分别获取UIAbility实例的上下文信息（UIAbilityContext），详细请参见[获取UIAbility的上下文信息](https://developer.huawei.com/consumer/cn/doc/harmonyos-guides/uiability-usage#获取uiability的上下文信息)。
