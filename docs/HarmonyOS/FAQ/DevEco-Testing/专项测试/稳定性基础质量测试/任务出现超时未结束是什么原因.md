# 任务出现超时未结束是什么原因

请检查电脑的睡眠设置。如果未设置为“从不睡眠”，可能会导致任务超时未完成。

在Windows中设置电源选项：打开控制面板，选择电源选项，点击更改计划设置，选择使计算机进入睡眠状态，设置为“从不”，然后保存修改。

在Mac上设置方法如下：进入系统偏好设置，选择电池或节能，将“此时间段后关闭显示器”设置为“永不”，并勾选“当显示器关闭时，防止电脑自动进入睡眠”。
