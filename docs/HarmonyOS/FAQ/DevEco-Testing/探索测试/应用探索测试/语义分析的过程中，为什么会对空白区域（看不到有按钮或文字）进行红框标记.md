# 语义分析的过程中，为什么会对空白区域（看不到有按钮或文字）进行红框标记

系统会根据当前页面的布局信息对控件进行标记。控件分为可见控件和不可见控件。空白区域内可能存在不可见控件，以确保测试的正常执行。
