# 测试过程中，为什么测试页面截图会出现页面切换状态

在遍历操作过程中，页面的滑动操作与截图操作不同步。测试截图时，有极小概率会截取到页面切换的过程，这是正常现象，不影响应用探索测试的整体功能。
