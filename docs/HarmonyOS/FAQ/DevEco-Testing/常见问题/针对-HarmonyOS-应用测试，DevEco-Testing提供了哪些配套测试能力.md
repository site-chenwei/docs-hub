# 针对 HarmonyOS 应用测试，DevEco Testing提供了哪些配套测试能力

已发布DevEco Testing Windows和Mac客户端，提供了专项测试、上架预检、探索测试、回归测试等测试服务，包括应用探索测试、场景化性能测试、回归测试、稳定性测试、性能测试、性能指标监控测试、UIViewer、设备投屏、应用图谱管理工具、性能报告自动分析、性能报告对比等。同时，提供DevEco Testing Hypium UI自动化测试框架及智能编码插件，覆盖应用测试全周期，助力打造高品质应用。
